Access the edit page on a bug by tacking a +edit to the end of the
bug's URL.

  >>> print http(r"""
  ... GET /malone/bugs/3/+edit HTTP/1.1
  ... Authorization: Basic Zm9vLmJhckBjYW5vbmljYWwuY29tOnRlc3Q=
  ... """)
  HTTP/1.1 200 Ok
  ...
  Content-Type: text/html;charset=utf-8
  ...
  <h3>...</h3>
