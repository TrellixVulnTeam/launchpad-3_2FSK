We do some special-handling of the cases where upstream doesn't use
Malone as their official bugtracker. In this situation, we want to tell
people what their options are. 

=== Upstreams ===

Alsa-utils, for instance, doesn't use Malone:

    >>> user_browser.open(
    ...     'http://bugs.launchpad.dev/alsa-utils/+filebug')
    >>> "does not use Malone" in user_browser.contents
    True
    >>> user_browser.getControl(name="field.title")
    Traceback (most recent call last):
      ..
    LookupError: name 'field.title'

But it's packaged in Ubuntu and Debian, and we suggest those packages
for filing bugs:

    >>> "Ubuntu alsa-utils" in user_browser.contents
    True

We don't know what bug tracker they use either:

    >>> "doesn't know what bug tracker" in user_browser.contents
    True

Gnomebaker doesn't use Malone either. We don't have packaging
information for it:

    >>> user_browser.open(
    ...     'http://bugs.launchpad.dev/gnomebaker/+filebug')
    >>> "does not use Malone" in user_browser.contents
    True
    >>> user_browser.getLink("telling Launchpad") is not None
    True

But we do know what tracker it uses:

    >>> user_browser.getLink("GnomeGBug GTracker") is not None
    True


== Advanced Filebug ==

The same happens with the advanced filebug form:

  >>> user_browser.open(
  ...     "http://launchpad.dev/products/alsa-utils/+filebug-advanced")
  >>> print user_browser.contents
  <!DOCTYPE...
  ...does not use Malone...
  ...

=== Distros ===

Distributions have less options available if they don't use Malone:

    >>> user_browser.open(
    ...     'http://bugs.launchpad.dev/debian/+filebug')
    >>> "does not use Malone" in user_browser.contents
    True

They get the same messages in the advanced filebug page:

  >>> user_browser.open(
  ...     "http://launchpad.dev/distros/debian/+filebug-advanced")
  >>> print user_browser.contents
  <!DOCTYPE...
  ...does not use Malone...
  ...

=== Distro Source Packages ===

It's also not possible to file a bug on any of the distribution's
source package.

  >>> user_browser.open(
  ...     "http://launchpad.dev/debian/+source/mozilla-firefox/"
  ...     "+filebug")
  >>> print user_browser.contents
  <!DOCTYPE...
  ...does not use Malone...
  ...

Not even using the advanced filebug page:

  >>> user_browser.open(
  ...     "http://launchpad.dev/distros/debian/+source/mozilla-firefox/"
  ...     "+filebug-advanced")
  >>> print user_browser.contents
  <!DOCTYPE...
  ...does not use Malone...
  ...
