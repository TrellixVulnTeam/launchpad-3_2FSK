This check will be sure that we are escaping correctly html tags
inside textareas.

  >>> user_browser.open(
  ...     'http://launchpad.dev/ubuntu/hoary/+source/pmount/+pots/pmount/hr/'
  ...     '+translate')
  >>> user_browser.url
  'http://launchpad.dev/ubuntu/hoary/+source/pmount/+pots/pmount/hr/+translate'

We are going to assign a value with html tags to see that we escape it when
rendered as part of a textarea.

  >>> textarea = user_browser.getControl(name='msgset_67_hr_translation_0_new')
  >>> textarea.value = 'Upotreba:\r\n\r\n%s [opcije] <uređaj> [<etiketa>]\r\n'
  >>> user_browser.getControl(name='submit_translations').click()
