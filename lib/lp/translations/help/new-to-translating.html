Tips for new translators

If you're new to translating using Launchpad, there are a few things you might find it useful to know.

  • You can tell Launchpad which languages you prefer to work in.
  • You can help any of hundreds of projects that use Launchpad to translate their software. Also, you can help translate distributions, such as Ubuntu.
  • Each project/distribution decides how to balance openness with the need for review of translations by selecting a permissions policy. For most projects, you'll be able to suggest translations that will then be reviewed by someone else.
