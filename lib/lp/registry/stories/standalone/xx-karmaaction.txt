= The actions that give karma to people =

The list of actions that may give karma to someone can be seen at
/karmaaction, but only LP admins can access that page.

    >>> user_browser.open('http://launchpad.dev/karmaaction')
    Traceback (most recent call last):
    ...
    Unauthorized:...launchpad.Admin...

    >>> admin_browser.open('http://launchpad.dev/karmaaction')
    >>> print extract_text(find_tag_by_id(admin_browser.contents, 'karmas'))
    Category        Action          Points
    bugs            Bug Accepted    5
    ...

From there you can edit any of those actions.

    >>> admin_browser.getLink('Bug Accepted').click()
    >>> admin_browser.title
    'Edit karma action'

    >>> admin_browser.getControl('Title').value = 'Bug accepted'
    >>> admin_browser.getControl('Points').value = '1'
    >>> admin_browser.getControl('Change').click()

    >>> admin_browser.title
    'Karma actions'
    >>> print extract_text(find_tag_by_id(admin_browser.contents, 'karmas'))
    Category        Action          Points
    bugs            Bug accepted    1
    ...
