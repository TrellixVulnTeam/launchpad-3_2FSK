When indicating that a bug affects upstream, the product and bugtracker
field is pre-populated with if a packaging link is found.

    >>> user_browser.open(
    ...  'http://launchpad.dev/distros/ubuntu/+source/mozilla-firefox/+bug/8')
    >>> user_browser.getLink('Upstream').click()
    >>> user_browser.getControl('Product').value
    'firefox'

Firefox uses Malone, so the bug tracker field is empty.
