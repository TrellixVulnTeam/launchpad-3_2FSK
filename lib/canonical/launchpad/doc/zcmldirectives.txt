= ZCML Directives =

We have a bunch of custom zcml directives in Launchpad.

== Facets and Links ==

BasicLink is one of the building-blocks of this directive.

  >>> from canonical.launchpad.webapp.metazcml import BasicLink
  >>> bugs = BasicLink('bugs', '+bugs', 'Bugs Application',
  ...   'The bug management application')
  >>> bugs.id
  'bugs'
  >>> bugs.href
  '+bugs'
  >>> bugs.title
  'Bugs Application'
  >>> bugs.summary
  'The bug management application'

  A BasicLink provides the IBasicLink interface.

  >>> from canonical.launchpad.interfaces import IBasicLink
  >>> IBasicLink.providedBy(bugs)
  True
  >>> from zope.interface.verify import verifyObject
  >>> verifyObject(IBasicLink, bugs)
  True

A Link is like a BasicLink, but with the addition of an attribute to say
if it is enabled or disabled.  A FacetList returns Links via its links
attribute.

  >>> from canonical.launchpad.webapp.metazcml import Link
  >>> from canonical.launchpad.interfaces import ILink
  >>> disabled_bugs = Link(bugs, False)
  >>> ILink.providedBy(disabled_bugs)
  True
  >>> verifyObject(ILink, disabled_bugs)
  True
  >>> disabled_bugs.id == bugs.id
  True
  >>> disabled_bugs.href == bugs.href
  True
  >>> disabled_bugs.title == bugs.title
  True
  >>> disabled_bugs.summary == bugs.summary
  True
  >>> disabled_bugs.enabled
  False
