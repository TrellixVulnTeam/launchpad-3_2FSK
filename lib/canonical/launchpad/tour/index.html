Bug tracking

Sharing code means sharing bugs.

Launchpad helps you work with different groups affected by the same bug. By sharing bug reports and their comments between communities you can develop a conversation across project boundaries to find and publicise a fix. Even where the bug is tracked elsewhere – such as in Bugzilla or Trac – Launchpad will import comments and status information directly into your bug report.

Code hosting and tracking

Democratise your code.

Launchpad and Bazaar reduce the barriers to contributing code to your project. Anyone can create their own version controlled branch of your code and publish it alongside your official branches. Publicly review new and vote on whether to merge new branches into your trunk. Import CVS and Subversion repositories into Bazaar branches.

Translations

Translators don’t have to be coders.

Launchpad’s intuitive web interface allows anyone to help translate your project, ready for export in standard GNU GetText files. 30,000 translators working in 257 languages, with automatic suggestions from a library of one million strings.
