Anonymous doesn't have access to the page:

  >>> anon_browser.handleErrors = True
  >>> anon_browser.open("http://launchpad.dev/ubuntu/hoary/+source/mozilla-firefox/+support-contact")
  >>> anon_browser.url
  'http://launchpad.dev/ubuntu/hoary/+source/mozilla-firefox/+support-contact/+login'

Any logged in person has access to the page.

  >>> browser.addHeader('Authorization', 'Basic foo.bar@canonical.com:test')
  >>> browser.open("http://launchpad.dev/ubuntu/hoary/+source/mozilla-firefox/+support-contact")
  >>> browser.url
  'http://launchpad.dev/ubuntu/hoary/+source/mozilla-firefox/+support-contact'

Add Foo Bar, Launchpad Developers and the Ubuntu Team as support
contacts.

  >>> browser.getControl("I want to be an answer contact for Hoary mozilla-firefox").click()
  >>> browser.getControl(name="field.support_contact_teams").value = ["ubuntu-team", "launchpad"]
  >>> browser.getControl("Continue").click()
  >>> browser.url
  'http://launchpad.dev/ubuntu/hoary/+source/mozilla-firefox/+tickets'

Check that the change worked:

  >>> from canonical.launchpad.database import Distribution, SourcePackageName, SupportContact
  >>> ubuntu = Distribution.selectOneBy(name="ubuntu")
  >>> mf = SourcePackageName.selectOneBy(name="mozilla-firefox")
  >>> [sc.person.name for sc in SupportContact.selectBy(distribution=ubuntu, sourcepackagename=mf)]
  [u'name16', u'launchpad', u'ubuntu-team']

Remove Foo Bar and the Ubuntu team as support contacts.

  >>> browser.open("http://launchpad.dev/ubuntu/hoary/+source/mozilla-firefox/+support-contact")
  >>> browser.getControl("I want to be an answer contact for Hoary mozilla-firefox").click()
  >>> browser.getControl(name="field.support_contact_teams").value
  ['launchpad', 'ubuntu-team']
  >>> browser.getControl(name="field.support_contact_teams").value = ["launchpad"]
  >>> browser.getControl("Continue").click()
  >>> browser.url
  'http://launchpad.dev/ubuntu/hoary/+source/mozilla-firefox/+tickets'

And check the outcome:

  >>> [sc.person.name for sc in SupportContact.selectBy(distribution=ubuntu, sourcepackagename=mf)]
  [u'launchpad']
