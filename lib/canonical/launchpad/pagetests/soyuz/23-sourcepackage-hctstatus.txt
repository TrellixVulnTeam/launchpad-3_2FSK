Packaging data for a source package can be viewed on the source
package's +hctstatus page.

  >>> from zope.testbrowser import Browser
  >>> browser = Browser()

  >>> browser.open("http://localhost:9000/distros/ubuntu/hoary/+source/mozilla-firefox/+hctstatus")
  >>> browser.contents
  '...<h2>Packaging data for Hoary mozilla-firefox</h2>...'
