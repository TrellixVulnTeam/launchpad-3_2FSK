DistroRelease information edit.
This tests makes a modification in warty distrorelease and check 
if the page is not broken.

Log in and navigate to the edit page.

  >>> from zope.testbrowser import Browser
  >>> browser = Browser()
  >>> browser.addHeader("Authorization", "Basic foo.bar@canonical.com:test")

  >>> browser.open('http://localhost:9000/distros/ubuntu/warty')
  >>> browser.getLink('Edit Details').click()
  >>> browser.url
  'http://localhost:9000/distros/ubuntu/warty/+edit'

Fill out the form and submit.

  >>> browser.getControl("Display name").value = 'WartyII'
  >>> browser.getControl("Title").value = 'The Warty Warthog II Release'
  >>> browser.getControl("Summary").value = 'summary'
  >>> browser.getControl("Description").value = 'description'
  >>> browser.getControl("Change").click()
  >>> browser.url
  'http://localhost:9000/distros/ubuntu/warty/+edit'
  >>> browser.contents
  '...Updated on...'
