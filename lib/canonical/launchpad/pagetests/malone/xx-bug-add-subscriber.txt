Non-Personal Subscriptions
==========================

In addition to subscribing / unsubscribing yourself, it is possible to add a
person as a subscriber to the bug. You would use this feature to subscribe
someone else to the bug.

First, let's make sure we can see the page.

    >>> user_browser.open('http://bugs.launchpad.dev/firefox/+bug/1/+addsubscriber')
    >>> user_browser.headers['status']
    '200 Ok'

Now, we'll subscribe sabdfl to the bug.

    >>> user_browser.open('http://bugs.launchpad.dev/firefox/+bug/1/+addsubscriber')
    >>> user_browser.getControl("Person").value = "sabdfl"
    >>> user_browser.getControl("Add").click()
    >>> user_browser.url
    'http://bugs.launchpad.dev/firefox/+bug/1'

Now, we should see Mark Shuttleworth as a subscriber.

    >>> from canonical.launchpad.ftests.test_pages import find_portlet
    >>> user_browser.open('http://bugs.launchpad.dev/firefox/+bug/1')
    >>> portlet = find_portlet(user_browser.contents, 'Subscribers to bug 1')
    >>> for ul in portlet('ul'):
    ...   print ul.renderContents()
    <BLANKLINE>
    ...Mark Shuttleworth...
