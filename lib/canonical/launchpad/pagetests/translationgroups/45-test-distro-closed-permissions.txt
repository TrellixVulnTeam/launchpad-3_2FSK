
So now Foo Bar is the Afrikaans translator for the PolyGlot translation
group, and they are the translation group for Ubuntu, which has Closed
translations. This means that Foo Bar should be able to translate any
strings in Ubuntu to Afrikaans. In other languages, Foo Bar should be
warned that he is not a designated translator.

Let's see if Foo Bar can see the translated strings in Southern Sotho. We
expect them to be told they can only make contributions:

  >>> print http(r"""
  ... GET /distros/ubuntu/hoary/+source/evolution/+pots/evolution-2.2/st/+translate HTTP/1.1
  ... Authorization: Basic Zm9vLmJhckBjYW5vbmljYWwuY29tOnRlc3Q=
  ... """)
  HTTP/1.1 200 Ok
  ...You are not an official translator for this file...
  ...

However, in Afrikaans, Foo Bar DOES have the ability to edit directly.

First, we show that "evolution minicard" is not translated.

  >>> print http(r"""
  ... GET /distros/ubuntu/hoary/+source/evolution/+pots/evolution-2.2/af/+translate HTTP/1.1
  ... Accept-Language: en-us,en;q=0.7,af;q=0.3
  ... Authorization: Basic Zm9vLmJhckBjYW5vbmljYWwuY29tOnRlc3Q=
  ... """)
  HTTP/1.1 200 Ok
  ...<input style="width: 100%;" type="text" lang="af"
            dir="ltr" name="msgset_137_af_translation_0"
            tabindex="22" />
  ...
