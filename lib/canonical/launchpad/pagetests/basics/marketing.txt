= Launchpad Marketing Microsite =

From Launchpad's front page, you can access a "marketing microsite"

    >>> browser.open('http://launchpad.dev/')
    >>> browser.getLink('Learn').click()
    >>> print browser.title
    About Launchpad
    >>> browser.url
    'http://launchpad.dev/+about'

That features three large tabs: "Home", "Get an account", and
"Take a tour".

    >>> navigation = find_tag_by_id(browser.contents, 'navigation')
    >>> home_link = navigation.find('a')
    >>> print home_link.renderContents()
    Home
    >>> print home_link['href']
    http://launchpad.dev/+about

    >>> account_link = browser.getLink('Get an account')
    >>> account_link.url
    'http://launchpad.dev/+about/+login'

    >>> tour_link = browser.getLink('Take a tour')
    >>> tour_link.url
    'http://launchpad.dev/+tour'

== About each application ==

The 'About Launchpad' page has one link to each Launchpad application
which brings the Overview page for that application. The overview page
has a link to the FAQs and Tour page for the application.

    >>> def test_app_marketing_pages(
    ...     name, start_page='http://launchpad.dev/+about'):
    ...     """Navigate from About Launchpad through the app marketing pages.
    ...
    ...     :param name: The name of the first link to follow.
    ...     :param start_page: The page where to start the navigation.
    ...     """
    ...     browser.open(start_page)
    ...     for link in [name, 'Application tour', 'FAQs', 'Overview']:
    ...         browser.getLink(link).click()
    ...         print "%s: %s" % (browser.title, browser.url)
