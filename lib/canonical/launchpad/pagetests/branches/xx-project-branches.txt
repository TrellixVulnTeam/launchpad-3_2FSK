= Project Branches Overview =

    >>> from canonical.launchpad.ftests.branch_helper import (
    ...     reset_all_branch_last_modified)
    >>> reset_all_branch_last_modified()


Projects link their branch listing page.

    >>> browser.open('http://launchpad.dev/mozilla')
    >>> browser.getLink('Code').click()
    >>> print browser.title
    The Mozilla Project's Bazaar branches registered in Launchpad


== Default page for code site ==

When going directly to a project on the code rootsite for launchpad,
the branch listing is the default page shown.

If there are branches, then they are displayed.

    >>> browser.open('http://code.launchpad.dev/mozilla')
    >>> print browser.title
    The Mozilla Project's Bazaar branches registered in Launchpad
    >>> table = find_tag_by_id(browser.contents, 'branchtable')
    >>> for row in table.tbody.fetch('tr'):
    ...     print extract_text(row)
    lp://dev/~name12/firefox/main            New   firefox ...
    lp://dev/~sabdfl/firefox/release--0.9.1  New   firefox ...
    lp://dev/~sabdfl/firefox/release-0.8     New   firefox ...
    lp://dev/~sabdfl/firefox/release-0.9     New   firefox ...
    lp://dev/~sabdfl/firefox/release-0.9.2   New   firefox ...
    lp://dev/~stevea/thunderbird/main        New   thunderbird ...

If there are not any branches, a relevant message is shown.

    >>> browser.open('http://code.launchpad.dev/aaa')
    >>> print browser.title
    The Test Project's Bazaar branches registered in Launchpad
    >>> message = find_tag_by_id(browser.contents, 'no-branch-message')
    >>> print extract_text(message)
    There are no branches registered for the Test Project in Launchpad...
