The bug task edit page now features a new and improved assignee
widget, which makes it easier to "take" a task.

  >>> print http(r"""
  ... GET /distros/debian/+sources/mozilla-firefox/+bug/1/+editstatus HTTP/1.1
  ... Authorization: Basic Zm9vLmJhckBjYW5vbmljYWwuY29tOnRlc3Q=
  ... Cookie: launchpad=cB8B19HiXp7whphyHXL-4zHCb5QJy.chSata4WiN2w3eRyOpyVk63w
  ... """)
  HTTP/1.1 200 Ok
  ...
  ...Assigned to:...
  ...nobody...
  ...me...
