= Answers Virtual Host =

The answers.launchpad.net domain is used for the Support Tracker
application. When visiting objects on that virtual host, the default
selected facet is the Support facet.

== Product ==

    >>> anon_browser.open('http://answers.launchpad.dev/firefox')
    >>> print anon_browser.title
    Support requests for Mozilla Firefox

== Distribution ==

    >>> anon_browser.open('http://answers.launchpad.dev/ubuntu')
    >>> print anon_browser.title
    Support requests for Ubuntu

== Soure Package ==

    >>> anon_browser.open(
    ...     'http://answers.launchpad.dev/ubuntu/+source/mozilla-firefox')
    >>> print anon_browser.title
    Support requests for mozilla-firefox in ubuntu

    >>> anon_browser.open(
    ...     'http://answers.launchpad.dev/ubuntu/hoary/+source/mozilla-firefox')
    >>> print anon_browser.title
    Support requests for Hoary mozilla-firefox

== Project ==

    >>> anon_browser.open(
    ...     'http://answers.launchpad.dev/mozilla')
    >>> print anon_browser.title
    Support requests for the Mozilla Project

== Person ==

    >>> anon_browser.open('http://answers.launchpad.dev/~name16')
    >>> print anon_browser.title
    Support requests involving Foo Bar
