A security contact gets all email about security-related bugs.

A security contact can be set on a distribution.

    >>> browser.addHeader("Authorization", "Basic foo.bar@canonical.com:test")

    >>> browser.open("http://localhost:9000/distros/ubuntu/+securitycontact")
    >>> browser.getControl("Security Contact").value = "name16"
    >>> browser.getControl("Continue").click()
    >>> print browser.contents
    <!DOCTYPE...
    ...Successfully changed the security contact to foo.bar@canonical.com...
    ...

Or on an upstream.

    >>> browser.open("http://localhost:9000/products/firefox/+securitycontact")
    >>> browser.getControl("Security Contact").value = "name21"
    >>> browser.getControl("Continue").click()
    >>> print browser.contents
    <!DOCTYPE...
    ...Successfully changed the security contact to Hoary Gnome Team...
    ...
