Each product in Launchpad has a Translations page.

    >>> browser.open('http://launchpad.dev/products/evolution/+translations')
    >>> browser.url
    'http://launchpad.dev/products/evolution/+translations'
    >>> 'Translations of The Evolution Groupware Application' in browser.contents
    True

If a product is not set up for translations in Rosetta, and you are its
registrant, the Translations page tells you how to proceed.

    >>> browser.addHeader('Authorization', 'Basic mark@hbd.com:test')
    >>> browser.open('http://launchpad.dev/products/alsa-utils/+translations')
    >>> print browser.contents
    <...
    ...How to set up translations...
    ...
    ...choose which product series...
    ...

If you're logged in as someone else, the Translations page suggests that you
contact the product's developers to set up translations.

    >>> browser.addHeader('Authorization', 'Basic no-priv@canonical.com:test')
    >>> browser.open('http://launchpad.dev/products/alsa-utils/+translations')
    >>> print browser.contents
    <...
    ...Not using Rosetta...
    ...
    ...contact the alsa-utils developers...
    ...

And if you're not logged in at all, you might or might not be the registrant,
so the text reflects both possibilities.

    >>> anon_browser.open('http://launchpad.dev/products/alsa-utils/+translations')
    >>> print browser.contents
    <...
    ...If you&#8217;re Mark Shuttleworth (the registrant of alsa-utils),
    .../products/alsa-utils/+login"...log in...
    ...
    ...Otherwise,...contact the alsa-utils developers...
    ...
