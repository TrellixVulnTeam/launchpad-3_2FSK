
Check that the index page for a POTemplate is listing all available
translations.

  >>> url = ("http://launchpad.dev/"
  ...     "ubuntu/hoary/+source/evolution/+pots/evolution-2.2/")
  >>> anon_browser.open(url)
  >>> anon_browser.contents
  '...
  ...Translation status for...
  ...Template "evolution-2.2" in Ubuntu Hoary package "evolution"...
  ...
  ...22 messages...
  ...
  ...Owner:...
  ...Rosetta Administrators...
  ...'

Check that the page is loading when you have more than one template inside the
same product.

  >>> url = ("http://launchpad.dev/"
  ...     "evolution/trunk/+pots/evolution-2.2-test")
  >>> anon_browser.open(url)
  >>> print find_main_content(anon_browser.contents)
  <...
        Other templates from the same source:
