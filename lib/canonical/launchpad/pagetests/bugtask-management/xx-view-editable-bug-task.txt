An anonymous user who tries to access the bugtask edit page will be
redirected to the login page.

    >>> browser.open("http://launchpad.dev/thunderbird/+bug/9/+editstatus")
    Traceback (most recent call last):
      ...
    Unauthorized: ...

(Even when the product has a bug supervisor:
https://launchpad.net/malone/+bug/49891)

    >>> from zope.component import getUtility
    >>> from canonical.database.sqlbase import flush_database_updates
    >>> from canonical.launchpad.ftests import login, logout
    >>> from canonical.launchpad.interfaces import IProductSet, IPersonSet

    >>> login("test@canonical.com")

    >>> firefox = getUtility(IProductSet).getByName("firefox")
    >>> from zope.security.proxy import removeSecurityProxy
    >>> sample_person = getUtility(IPersonSet).getByName("name12")
    >>> firefox = removeSecurityProxy(firefox).master
    >>> sample_person = removeSecurityProxy(sample_person).master
    >>> firefox.setBugSupervisor(sample_person.master, sample_person.master)

    >>> flush_database_updates()

    >>> logout()

    >>> browser.open("http://launchpad.dev/firefox/+bug/1/+editstatus")
    Traceback (most recent call last):
      ...
    Unauthorized: ...

Any logged-in user can edit a bugtask.

    >>> browser = setupBrowser(auth="Basic test@canonical.com:test")
    >>> browser.open("http://launchpad.dev/firefox/+bug/6/+editstatus")
    >>> browser.title
    'Bug #6 in Mozilla Firefox...'
