From the distribution task page, you can access the release targeting
screen to select specific releases to which a bugfix should be
targeted.

  >>> print http(r"""
  ... GET /distros/debian/+source/mozilla-firefox/+bug/1/+target HTTP/1.1
  ... Authorization: Basic dGVzdEBjYW5vbmljYWwuY29tOnRlc3Q=
  ... """)
  HTTP/1.1 200 Ok
  ...

To target to a specific release, select the checkbox beside that
release.

  >>> print http(r"""
  ... POST /distros/debian/+source/mozilla-firefox/+bug/1/+target HTTP/1.1
  ... Accept-Language: en-ca,en-us;q=0.8,en;q=0.5,fr-ca;q=0.3
  ... Authorization: Basic dGVzdEBjYW5vbmljYWwuY29tOnRlc3Q=
  ... Content-Length: 72
  ... Content-Type: application/x-www-form-urlencoded
  ... 
  ... target=woody.mozilla-firefox&savetargets=Target+Fix+to+Selected+Releases""")
  HTTP/1.1 303 See Other
  ...

You are brought back to the task page. The newly targeted bug task is
shown in the "also reported in" portlet.

  >>> print http(r"""
  ... GET /distros/debian/+source/mozilla-firefox/+bug/1/+editstatus HTTP/1.1
  ... Accept-Language: en-ca,en-us;q=0.8,en;q=0.5,fr-ca;q=0.3
  ... Authorization: Basic dGVzdEBjYW5vbmljYWwuY29tOnRlc3Q=
  ... Cookie: launchpad=2RlITCZlIZAagraDp8iQ-00xVfInmZ8u45u9mbE2n1S0ApJdWxVP.0
  ... """)
  HTTP/1.1 200 Ok
  ...
  ...Also reported in:...
  ...mozilla-firefox (Debian Woody)...
  ...
