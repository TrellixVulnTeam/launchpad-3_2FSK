The bug listing pages save their search criteria.

    >>> browser.open("http://localhost:9000/distros/ubuntu/+bugs")

    >>> '4 results' in browser.contents
    True

If, for example, you click on one of the canned search links:

    >>> browser.getLink("Unconfirmed").click()

The result set is filtered to show only Unconfirmed bugs.

    >>> '3 results' in browser.contents
    True

    >>> print browser.contents
    <!DOCTYPE...
    ...Firefox does not support SVG...
    ...another test bug...
    ...Blackhole Trash folder...

Changing the sort order of the results preserves the filter.

    >>> browser.getControl(name="orderby").value = ['targetname']
    >>> browser.getControl(name="search").click()

    >>> '3 results' in browser.contents
    True

    >>> print browser.contents
    <!DOCTYPE...
    ...Blackhole Trash folder...
    ...another test bug...
    ...Firefox does not support SVG...
