We should be able to edit the details on a sprint. We will log in as Sample
Person and edit the ubz sprint.

  >>> user_browser.open('http://launchpad.dev/sprints/ubz/')
  >>> user_browser.title
  'Ubuntu Below Zero (sprint or meeting)'

  >>> address = 'Holiday Inn Select, Downtown Montreal, Canada'
  >>> address in user_browser.contents
  False
  >>> user_browser.getLink('Edit Details').click()
  >>> user_browser.url
  'http://launchpad.dev/sprints/ubz/+edit'

And when we POST to that form, we expect to be redirected to the sprint home
page.

  >>> user_browser.getControl('Meeting Address').value = address
  >>> user_browser.getControl('Change').click()

  >>> user_browser.url
  'http://launchpad.dev/sprints/ubz'

  >>> address in user_browser.contents
