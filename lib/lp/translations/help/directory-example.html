Example directory layout for translation files

Each template (.pot) file must be in its own directory. For example:

  • template1/template1.pot
    • template1/de.po
    • template1/fr.po
  • template2/template2.pot
    • template2/de.po
    • template2/fr.po

You should follow Launchpad translation import guidelines to ensure your translation files are imported successfully.
