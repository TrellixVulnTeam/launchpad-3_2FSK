Each of the +bugs bug listings have a sort order widget. Let's
demonstrate this functionality on each +bugs page.

1. The upstream +bugs page.

Default view:

    >>> browser.open("http://localhost:9000/products/firefox/+bugs")

    >>> print browser.contents
    <!DOCTYPE...
    ...Firefox install instructions should be complete...
    ...Reflow problems with complex page layouts...
    ...Firefox does not support SVG...

Newest:

    >>> browser.getControl(name="orderby").value = ["datecreated"]
    >>> browser.getControl("Search").click()

    >>> print browser.contents
    <!DOCTYPE...
    ...Firefox does not support SVG...
    ...Reflow problems with complex page layouts...
    ...Firefox install instructions should be complete...

2. The distro +bugs page.

Default view:

    >>> browser.open("http://localhost:9000/distros/debian/+bugs")
    >>> print browser.contents
    <!DOCTYPE...
    ...Bug Title Test...
    ...Firefox does not support SVG...
    ...Blackhole Trash folder...

By recently changed:

    >>> browser.getControl(name="orderby").value = ["-date_last_updated"]
