= Launchpad Marketing Microsite =

From Launchpad's front page, you can access a "marketing microsite"

    >>> browser.open('http://launchpad.dev/')
    >>> browser.getLink('Take a tour').click()
    >>> print browser.title
    Launchpad tour
    >>> browser.url
    'http://launchpad.dev/+tour/index.html'

The tour is circular. Clicking the Next button repeatedly will bring you
back to the tour start.

    >>> def take_the_tour(steps_taken=0):
    ...     browser.getLink('Next').click()
    ...     print browser.url
    ...     if browser.url != 'http://launchpad.dev/+tour/index.html':
    ...         if steps_taken >= 20:
    ...             raise RuntimeError('Never ending tour!')
    ...         take_the_tour(steps_taken=steps_taken+1)
    >>> take_the_tour()
    http://launchpad.dev/+tour/bugs.html
    http://launchpad.dev/+tour/branch-hosting-tracking.html
    http://launchpad.dev/+tour/code-review.html
    http://launchpad.dev/+tour/translation.html
    http://launchpad.dev/+tour/community.html
    http://launchpad.dev/+tour/ppa.html
    http://launchpad.dev/+tour/community-support.html
    http://launchpad.dev/+tour/feature-tracking.html
    http://launchpad.dev/+tour/release-management.html
    http://launchpad.dev/+tour/api.html
    http://launchpad.dev/+tour/index.html

The images from the tour are retrieved relative to +tour.

    >>> browser.open('http://launchpad.dev/+tour/images/home/main-image.jpg')
    >>> browser.open('http://launchpad.dev/+tour/images/btn-next.png')

But the source directory isn't available:

    >>> browser.open(
    ...     'http://launchpad.dev/+tour/source/code-hosting_SVG.svg')
    Traceback (most recent call last):
      ...
    NotFound:...


== About each application ==

The 'About Launchpad' page has one link to each Launchpad application
which brings the Overview page for that application. The overview page
has a link to the FAQs and Tour page for the application.

    >>> def test_app_marketing_pages(name):
    ...     """Navigate from About Launchpad through the app marketing pages.
    ...
    ...     :param name: The name of the application to test.
    ...     """
    ...     browser.open('http://launchpad.dev/+about')
    ...     links = [name, '%s tour' % name, '%s FAQs' % name,
    ...              '%s overview' % name]
    ...     for link in links:
    ...         browser.getLink(link).click()
    ...         print "%s: %s" % (browser.title, browser.url)
