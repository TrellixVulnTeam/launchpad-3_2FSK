== Package Maintenance Report ==

From the main person page, the 'Packages' link will display the person's
Package Maintenance Report.


    >>> anon_browser.open('http://launchpad.dev/~sabdfl')
    >>> anon_browser.getLink('List assigned packages').click()
    >>> print anon_browser.title
    Packages maintained by Mark Shuttleworth
    >>> print anon_browser.url
    http://launchpad.dev/~sabdfl/+packages

That page display two tables, the list of 'Maintained'packages as
well as the list of 'Uploaded' packages. The table also shows the
number of open bugs and questions filed against each package.

    >>> maintained = find_tag_by_id(anon_browser.contents, 'maintained')
    >>> print extract_text(maintained)
    Name        Uploaded to Version  Date   Failures    Bugs  Questions
    alsa-utils  Debian Sid  1.0.9a-4 ...    None        0     0
    ...
    alsa-utils  Ubuntu Warty 1.0.8-1ubuntu1 ...None     0     0
    mozilla-firefox Ubuntu Hoary 0.9 ...    None        1     0
    ...

    >>> uploaded = find_tag_by_id(anon_browser.contents, 'uploaded')
    >>> print extract_text(uploaded)
    Name   Uploaded to     Version     Date    Failures    Bugs    Questions
    ...
    foobar  Ubuntu Breezy-autotest 1.0  ...    i386        0       0
    ...
    alsa-utils Ubuntu Hoary 1.0.9a-4ubuntu1... Not yet built 0     0...
