A source package bug task describes where in a given distro a bug
needs to be fixed.

  >>> print http(r"""
  ... GET /distros/ubuntu/+source/mozilla-firefox/+bug/1/+editstatus HTTP/1.1
  ... Authorization: Basic dGVzdEBjYW5vbmljYWwuY29tOnRlc3Q=
  ... """)
  HTTP/1.1 200 Ok
  ...
            <input type="text" name="field.sourcepackagename"
  ...
