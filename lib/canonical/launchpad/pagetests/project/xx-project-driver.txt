= Appointing a Project Driver =

Appoint Sample Person as the driver of the GNOME Project.

  >>> browser.addHeader('Authorization', 'Basic test@canonical.com:test')
  >>> browser.open('http://launchpad.dev/gnome')
  >>> browser.getLink('Appoint driver').click()

  >>> browser.url
  'http://launchpad.dev/gnome/+driver'
  >>> browser.title
  'Appoint the driver for The GNOME Project'

  >>> browser.getControl('Driver').value = 'name12'
  >>> browser.getControl('Change').click()

After changing the driver, the user is redirected to the overview page where a 
message inform him of the driver change.

  >>> browser.url
  'http://launchpad.dev/gnome'

  >>> for tag in find_tags_by_class(browser.contents, 'informational'):
  ...     tag.renderContents()
  'Successfully changed the driver to test@canonical.com'

Sample Person is listed as the driver of the project.

  >>> for tag in find_tags_by_class(browser.contents, 'summary'):
  ...     tag.find(text='Driver:').findNext('a').string
  u'Sample Person'
