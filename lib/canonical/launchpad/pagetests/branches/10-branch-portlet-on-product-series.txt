= Branch Details on Product Series =

== No links ==

If the product series has no associated branch then the 
revision control portlet should show 'None given'

  >>> user_browser.open('http://launchpad.dev/products/firefox/trunk')
  >>> user_browser.contents
  '<!DOCTYPE...
  ...Revision control details...
  ...<i>None given</i>...'
  
== Add association ==

Now if we associate a branch it should be there

  >>> user_browser.getLink('Change Series Details').click()
  >>> user_browser.url
  'http://launchpad.dev/products/firefox/trunk/+edit'
  >>> user_browser.getControl('Branch').value = '~name12/firefox/main'
  >>> user_browser.getControl('Change').click()

Should now be back at the main series page
 
  >>> user_browser.url
  'http://launchpad.dev/products/firefox/trunk'
  >>> user_browser.contents
  '<!DOCTYPE...
  ...Revision control details...
  ...<a href="http://launchpad.dev/people/name12/+branch/firefox/main">Mozilla Firefox Mainline</a>...'
