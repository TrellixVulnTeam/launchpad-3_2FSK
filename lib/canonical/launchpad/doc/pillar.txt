A few of the PillarsOfLaunchpad (tm) share a namespace to allow their name
to unambiguously refer to it. This allows us to make email interfaces
easier to use and to shorten some of our URLs. Currently, the objects that
share their name namespace are Product, Project and Distribution.

>>> from canonical.launchpad.interfaces import (
...     IPillarSet, IProduct, IProject, IDistribution
...     )
>>> pillar_set = getUtility(IPillarSet)

>>> 'ubuntu' in pillar_set
True
>>> pillar_set['ubuntu'].name
u'ubuntu'
>>> IDistribution.providedBy(pillar_set['ubuntu'])
True

>>> 'ubuntu-product' in pillar_set
True
>>> pillar_set['ubuntu-product'].name
u'ubuntu-product'
>>> IProduct.providedBy(pillar_set['ubuntu-product'])
True

>>> 'ubuntu-project' in pillar_set
True
>>> pillar_set['ubuntu-project'].name
u'ubuntu-project'
>>> IProject.providedBy(pillar_set['ubuntu-project'])
True

>>> 'fnord' in pillar_set
False
>>> pillar_set['fnord']
Traceback (most recent call last):
...
NotFoundError: 'fnord'

We can also use PillarSet to do a search across some of our pillars. Right now
this search is done across Products, Projects and Distributions.

>>> for row in pillar_set.search('mozilla'):
...     type, id, name, title, description, rank = row
...     print "%s: %s (%s) [relevance: %s]" % (type, title, name, rank)
project: The Mozilla Project (mozilla) [relevance: 10000000.0]
product: Mozilla Firefox (firefox) [relevance: 0.686029]
product: Mozilla Thunderbird (thunderbird) [relevance: 0.686029]
