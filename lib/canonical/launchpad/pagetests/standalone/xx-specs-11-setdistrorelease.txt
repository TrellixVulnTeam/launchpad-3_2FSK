It is possible to set a specification to a specific distribution
release.
This pagetest tries to add a spec to the Ubuntu distribution and set it
to the Grumpy distrorelease.

First we try to access the addspec page.

  >>> print http(r"""
  ... GET /distros/ubuntu/+addspec HTTP/1.1
  ... Authorization: Basic dGVzdEBjYW5vbmljYWwuY29tOnRlc3Q=
  ... """)
  HTTP/1.1 200 Ok
  ...
  ...<h1>Register a feature specification</h1>...
  ...

Now we try to open the setdistrorelease page, where there is a form to
target the newly created spec to a distribution.


  >>> print http(r"""
  ... GET /distros/ubuntu/+spec/testspec/+setdistrorelease HTTP/1.1
  ... Authorization: Basic dGVzdEBjYW5vbmljYWwuY29tOnRlc3Q=
  ... """)
  HTTP/1.1 200 Ok
  ...<h1>Target to a Distribution Release</h1>...
  ...
  
We'll fill the form, targeting the specification to the Grumpy
distrorelease.

  >>> print http(r"""
  ... POST /distros/ubuntu/+spec/testspec/+setdistrorelease HTTP/1.1
  ... Authorization: Basic dGVzdEBjYW5vbmljYWwuY29tOnRlc3Q=
  ... Content-Length: 600
  ... Content-Type: multipart/form-data; boundary=---------------------------13150465521855597243975128072
