Pagetests that use launchpadlib
===============================

As an alternative to crafting HTTP requests with the 'webservice'
object, you can write pagetests using launchpadlib.

  >>> from launchpadlib.testing.helpers import salgado_with_full_permissions
  >>> from launchpadlib.launchpad import Launchpad

  >>> launchpad = Launchpad.login(
  ...       'launchpad-library', 'salgado-change-anything', 'test',
  ...       service_root='http://api.launchpad.dev/', version="devel")

  >>> print launchpad.me.name
  salgado
