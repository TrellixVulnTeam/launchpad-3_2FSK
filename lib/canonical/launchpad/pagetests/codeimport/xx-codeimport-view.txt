For now, we put a link on the `code` page:

  >>> browser.open('http://code.launchpad.dev/')
  >>> browser.getLink('View all imports').click()
  >>> browser.url
  'http://code.launchpad.dev/+code-imports'

There are two CodeImports in the sample data:

  >>> from zope.component import getUtility
  >>> from canonical.launchpad.interfaces import ICodeImportSet
  >>> from canonical.lp.dbschema import RevisionControlSystems
  >>> from canonical.launchpad.interfaces import IProductSeriesSet
  >>> from canonical.launchpad.ftests import login, logout, ANONYMOUS
  >>> login(ANONYMOUS)
  >>> codeimportset = getUtility(ICodeImportSet)
  >>> codeimports = list(codeimportset.getAll())
  >>> len(codeimports)
  2
  >>> sorted(ci.name for ci in codeimports)
  [u'sample-cvs', u'sample-svn']
  >>> svn_import ,= [ci for ci in codeimports if ci.name==u'sample-svn']
  >>> cvs_import ,= [ci for ci in codeimports if ci.name==u'sample-cvs']
  >>> logout()

And they both show up in the page:

  >>> [(ci.name in browser.contents) for ci in codeimports]
  [True, True]

If we click on the code import's name, we go to the page for that
import:

  >>> browser.getLink('sample-svn').click()
  >>> browser.url
  'http://code.launchpad.dev/+code-imports/sample-svn'

It displays some information about the code import, for example the
URL of the Subversion branch the import is from:

  >>> svn_import.svn_branch_url in browser.contents
  True

And similar for the other import:

  >>> browser.getLink('Code Imports').click()
  >>> browser.getLink('sample-cvs').click()
  >>> browser.url
  'http://code.launchpad.dev/+code-imports/sample-cvs'
  >>> (cvs_import.cvs_root in browser.contents) and (cvs_import.cvs_module in browser.contents)
  True
