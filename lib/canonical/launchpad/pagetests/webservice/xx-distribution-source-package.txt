= Distribution Source Packages =

Source packages can be obtained from the context of a distribution.

    >>> ubuntu = webservice.get('/ubuntu').jsonBody()

    >>> alsa_utils = webservice.named_get(
    ...     ubuntu['self_link'], 'getSourcePackage',
    ...     name='alsa-utils').jsonBody()

    >>> print pretty(alsa_utils)
    {u'binary_package_names': u'',
     u'bug_reporting_guidelines': None,
     u'display_name': u'alsa-utils in ubuntu',
     u'distribution_link': u'http://.../ubuntu',
     u'name': u'alsa-utils',
     u'resource_type_link': u'http://.../#distribution_source_package',
     u'self_link': u'http://.../ubuntu/+source/alsa-utils',
     u'title': u'\u201calsa-utils\u201d source package in Ubuntu'}
