When indicating that a bug affects upstream, the product and bugtracker
fields is pre-populated with initial values if a packaging link is found.

    >>> user_browser.open(
    ...  'http://launchpad.dev/distros/ubuntu/+source/mozilla-firefox/+bug/8')
    >>> user_browser.getLink('Upstream').click()
    >>> user_browser.getControl('Product').value
    'firefox'

Firefox uses Malone, so the bug tracker field is empty.
(XXX: Can't check getControl(..).value, since it returns ['3'] even
      though nothing is selected. -- Bjorn Tillenius, 2006-09-12

    >>> print user_browser.contents
    <!...
    <select id="field.bugtracker" ... size="5" ...>
    <option value="3">Debian Bug tracker</option>
    <option value="2">GnomeGBug GTracker</option>
    <option value="1">The Mozilla.org Bug Tracker</option>
    <option value="4">The Ubuntu.com bugzilla</option>
    ...


If the package is linked to a product which as an external bug tracker
set, like Netapplet, both the product and the bug tracker widgets will
have suggested initial values.

    >>> user_browser.open(
    ...     'http://launchpad.dev/distros/ubuntu/+source/mozilla-firefox/'
    ...     '+bug/8/+distrotask')
    >>> user_browser.getControl('Distribution').value = ['ubuntu']
    >>> user_browser.getControl('Source Package').value = 'netapplet'
    >>> user_browser.getControl('Continue').click()
    >>> user_browser.url
    'http://launchpad.dev/distros/ubuntu/+source/netapplet/+bug/8'

    >>> user_browser.getLink('Upstream').click()
    >>> user_browser.getControl('Product').value
    'netapplet'
    >>> user_browser.getControl('Remote Bug Tracker').displayValue
    ['GnomeGBug GTracker']


If the source package has no upstream link. like thunderbird in Ubuntu,
then the product widget will have no initial value.

    >>> user_browser.open(
    ...     'http://launchpad.dev/distros/ubuntu/+source/mozilla-firefox/'
    ...     '+bug/8/+distrotask')
    >>> user_browser.getControl('Distribution').value = ['ubuntu']
    >>> user_browser.getControl('Source Package').value = 'thunderbird'
    >>> user_browser.getControl('Continue').click()
    >>> user_browser.url
    'http://launchpad.dev/distros/ubuntu/+source/thunderbird/+bug/8'

    >>> user_browser.getLink('Upstream').click()
    >>> user_browser.getControl('Product').value
    ''
