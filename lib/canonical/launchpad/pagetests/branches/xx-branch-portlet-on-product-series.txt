= Branch Details on Product Series =

== No links ==

If the product series has no associated branch then the
revision control portlet should show 'None given'

  >>> anon_browser.open('http://launchpad.dev/firefox/trunk')
  >>> print find_portlet(anon_browser.contents, 'Revision control details')
  <div class="portlet">
  ...<i>None given</i>...
  </div>


== Add association ==

Now if the registrant associates a branch it should be there:

  >>> browser.addHeader('Authorization', 'Basic test@canonical.com:test')
  >>> browser.open('http://launchpad.dev/firefox/trunk')
  >>> browser.getLink('Change Series Details').click()
  >>> browser.url
  'http://launchpad.dev/firefox/trunk/+edit'
  >>> browser.getControl('Branch').value = '~name12/firefox/main'
  >>> browser.getControl('Change').click()

Should now be back at the main series page

  >>> browser.url
  'http://launchpad.dev/firefox/trunk'
  >>> print find_portlet(browser.contents, 'Revision control details')
  <div class="portlet">
  ...<a href="http://launchpad.dev/~name12/+branch/firefox/main">Mozilla Firefox Mainline</a>...
  </div>
