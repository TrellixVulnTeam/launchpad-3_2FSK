= Renewing a member's subscription =

Renew the subscription of Sabdfl on team 'Ubuntu Gnome Team', which is
expired.

  >>> print http(r"""
  ... GET /~name18/+member/sabdfl HTTP/1.1
  ... Authorization: Basic Zm9vLmJhckBjYW5vbmljYWwuY29tOnRlc3Q=
  ... """)
  HTTP/1.1 200 Ok
  Content-Length: ...
  Content-Type: text/html;charset=utf-8
  ...
  ...Mark Shuttleworth...
  ...
  ...Expired...
  ...


  >>> print http(r"""
  ... POST /~name18/+member/sabdfl HTTP/1.1
  ... Authorization: Basic Zm9vLmJhckBjYW5vbmljYWwuY29tOnRlc3Q=
  ... Content-Length: ...
  ... Content-Type: application/x-www-form-urlencoded
  ... 
  ... editinactive=1&day=16&month=4&year=2030&comment=&renew=Renew""")
  HTTP/1.1 303 See Other
  ...
  Location: http://localhost:.../~name18/+members
  ...


  >>> print http(r"""
  ... GET /~name18/+members/ HTTP/1.1
  ... """)
  HTTP/1.1 200 Ok
  Content-Length: ...
  Content-Type: text/html;charset=utf-8
  ...
  ...Active members...
  ...
  ...Mark Shuttleworth...
  ...
  ...Approved...
  ...


Let's make sure that Sabdfl is now an APPROVED member of this team.

  >>> from canonical.launchpad.database import Person, TeamMembership
  >>> sabdfl = Person.byName('sabdfl')
  >>> team = Person.byName('name18')
  >>> tm = TeamMembership.selectOneBy(personID=sabdfl.id, teamID=team.id)
  >>> tm.statusname
  'Approved'
  >>> sabdfl.inTeam(team)
  True
