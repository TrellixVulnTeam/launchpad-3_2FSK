When a bug is marked as a duplicate of another bug, the main bug page
will show such information in a prominent way.

  >>> print http(r"""
  ... GET /debian/+source/mozilla-firefox/+bug/2 HTTP/1.1
  ... """)
  HTTP/1.1 200 Ok
  ...
        This report is a duplicate of
        <a href="...">bug #1</a>
  ...
