The bug index is the main page of a bug report. It contains all sorts
of interesting things related to fixing a problem: the bug
description, comments, external references, details of places in which
this bug needs to be fixed, and so forth.

  >>> print http(r"""
  ... GET /debian/+source/mozilla-firefox/+bug/2 HTTP/1.1
  ... """)
  HTTP/1.1 200 Ok
  ...Blackhole Trash folder...

If you are logged in, each "Affects" row contains a form for editing the status
in that context. The forms are hidden using CSS, but this test should still find
them.
