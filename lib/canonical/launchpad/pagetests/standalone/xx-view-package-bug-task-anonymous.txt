Anonymous users can view bug tasks on public bugs. Even though the URL
ends in +edit, this is in fact a read-only UI to anonymous users.

  >>> print http(r"""
  ... GET /distros/debian/+bugs/3/+edit HTTP/1.1
  ... """) #doctest: -NORMALIZE_WHITESPACE
  HTTP/1.1 200 Ok
