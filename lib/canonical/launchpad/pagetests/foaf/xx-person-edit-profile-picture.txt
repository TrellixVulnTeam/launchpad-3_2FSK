== Changing a person's profile picture ==

Users can change their profile picture on their Edit page.

Note that we have chosen not to expose the ability to customise a user's icon
or logo.

    >>> browser = setupBrowser(auth='Basic mark@hbd.com:test')
    >>> browser.open('http://launchpad.dev/~sabdfl')
    >>> browser.url
    'http://launchpad.dev/~sabdfl'

    # XXX mars 2008-06-11
    # Use direct navigation for the LP 2.0 transition. See bug #239192.
    >>> browser.open('http://launchpad.dev/~sabdfl/+edit2.0')
    >>> browser.url
    'http://launchpad.dev/~sabdfl/+edit2.0'

    >>> find_tag_by_id(browser.contents, 'field.mugshot_current_img').get('src')
    u'/@@/person-mugshot'

    >>> from canonical.launchpad.ftests import set_branding
    >>> set_branding(browser, icon=False, logo=False)

    >>> browser.getControl('Save Changes').click()

Here we see the updated values.

    >>> browser.url
    'http://launchpad.dev/~sabdfl'

    # XXX mars 2008-06-11
    # Use direct navigation for the LP 2.0 transition. See bug #239192.
    >>> browser.open('http://launchpad.dev/~sabdfl/+edit2.0')
    >>> browser.url
    'http://launchpad.dev/~sabdfl/+edit2.0'

    >>> browser.getControl(name='field.mugshot.action').value
    ['keep']
    >>> find_tag_by_id(browser.contents, 'field.mugshot_current_img').get('src')
    u'.../mugshot.png'
