= Adding a comment when not in a bug context =

Even if a bug isn't reported on a product, we can still view the bug in
that context. For example, bug #3 isn't reported on Evolution, but we
can still access bug #3 in the Evolution context.

    >>> user_browser.open('http://launchpad.dev/products/evolution/+bug/3')
    >>> user_browser.title
    '...Bug #3 is not in Evolution...'
    >>> for message in find_tags_by_class(user_browser.contents, 'message'):
    ...     print extract_text(message.renderContents())
    This bug isn't linked to Evolution.
    Would you like to report it here as well?

We can still add a comment to the bug.

    >>> user_browser.getControl(name='field.comment').value = 'A new comment.'
    >>> user_browser.getControl('Save Changes', index=-1).click()

    >>> comment_divs = find_tags_by_class(
    ...     user_browser.contents, 'boardCommentBody')
    >>> print comment_divs[-1].div.renderContents()
    <p>A new comment.</p>

There's also a button on the page to report the bug in the current location.

    >>> user_browser.getControl('Yes, report in Evolution').click()
    >>> user_browser.getLink('Bug #3').click()
    >>> print user_browser.title
    Bug #3 in mozilla-firefox...
