= Form Layout =

LaunchpadFormView based forms use a tabular layout for the widgets.
In most cases, the label is shown to the left of the actual widget,
but there are some exceptions:

 * multiline text widgets have the label shown above, and take up the
   full width of the form.

 * checkbox widgets have the label shown to the right.


== Normal Widgets ==

The new product form contains an example of a normal text widget:

  >>> browser.addHeader('Authorization', 'Basic test@canonical.com:test')
  >>> browser.open('http://launchpad.dev/products/+new')
  >>> content = find_main_content(browser.contents)

  >>> print content
  <...
  <tr>
    <th>
      <label for="field.displayname">Display Name:</label>
    </th>
    <td width="100%">
      <div>
        <input ... name="field.displayname" ... />
      </div>
      <p class="formHelp">...</p>
    </td>
  </tr>
  ...

If the text field is optional, then that is noted after the widget:

  >>> print content
  <...
  <tr>
    <th>
      <label for="field.homepageurl">Homepage URL:</label>
    </th>
    <td width="100%">
      <div>
        <input ... name="field.homepageurl" ...  />
        <span class="fieldRequired">(Optional)</span>
      </div>
      <p class="formHelp">...</p>
    </td>
  </tr>
  ...


== Multi-line Text Widgets ==

A multi-line text widget is displayed in two table rows:

  >>> print content
  <...
  <tr>
    <th colspan="2" style="text-align: left"><label
       for="field.summary">Summary:</label>
    </th>
  </tr>
  <tr>
    <td colspan="2">
      <div><textarea ... name="field.summary" ...></textarea></div>
      <p class="formHelp">...</p>
    </td>
  </tr>
  ...

If the multi-line text widget is optional, this is mentioned on the
label row:

  >>> print content
  <...
  <tr>
    <th colspan="2" style="text-align: left"><label
       for="field.description">Description:</label>
       <span class="fieldRequired">(Optional)</span>
    </th>
  </tr>
  <tr>
    <td colspan="2">
      <div><textarea ... name="field.description" ...></textarea></div>
      <p class="formHelp">...</p>
    </td>
  </tr>
  ...


== Checkbox Widgets ==

Checkboxes have their label to the right. One example is the checkbox for
specifying whether your project uses Launchpad Translations.

  >>> browser.open('http://launchpad.dev/evolution/+edit')
  >>> content = find_main_content(browser.contents)

  >>> print content
  <...
  <tr>
    <th></th>
    <td width="100%">
      ...
      <input ... name="field.official_rosetta" type="checkbox" ... />
      <label for="field.official_rosetta">Translations...</label>
    </td>
  </tr>
  ...
