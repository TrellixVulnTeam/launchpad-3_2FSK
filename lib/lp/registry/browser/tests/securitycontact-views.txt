HasSecurityContect
==================

The +edit view allows the project owner and admin to change the security
contact or the project. The view provides a label, page_title and cancel_url.

    >>> product = factory.makeProduct(name='bonkers', displayname='Bonkers')
    >>> view = create_view(product, name='+securitycontact')
    >>> print view.label
    Edit Bonkers security contact

    >>> print view.page_title
    Edit Bonkers security contact

    >>> print view.cancel_url
    http://launchpad.dev/bonkers

The view allows the owner or admin to change the security_contact field.

    >>> view.field_names
    ['security_contact']

    >>> print product.security_contact
    None

    >>> login_person(product.owner)
    >>> form = {
    ...     'field.security_contact': 'no-priv',
    ...     'field.actions.change': 'Change',
    ...     }
    >>> view = create_initialized_view(
    ...     product, name='+securitycontact', form=form)
    >>> view.errors
    []

    >>> print view.next_url
    http://launchpad.dev/bonkers

    >>> print product.security_contact.name
    no-priv
