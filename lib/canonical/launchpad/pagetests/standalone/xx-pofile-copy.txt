We are going to add a new translation to the message #130, it should remain
in the form once we do the copy.

Firts, we see that the message we are going to translate is empty.

  >>> user_browser.getControl(name='msgset_130_es_translation_0_new').value
  ''

And now, we change it.

  >>> user_browser.getControl(
  ...     name='msgset_130_es_translation_0_new').value = (
  ...         u'You will not erase me!')
  >>> user_browser.getControl(
  ...     name='msgset_130_es_translation_0_new_checkbox').value = True

Now, we are going to do a copy submission using current translation for
another message in the form.

  >>> user_browser.getControl(name='msgset_134_es_translation_0_copy').click()
  >>> user_browser.url
  'http://launchpad.dev/distros/ubuntu/hoary/+source/evolution/+pots/evolution-2.2/es/+translate'

And now, the new translation textarea has current translation:

  >>> user_browser.getControl(name='msgset_134_es_translation_0_new').value
  '\n caratas'

Its checkbox is set

  >>> user_browser.getControl(
  ...     name='msgset_134_es_translation_0_new_checkbox').value
  True

And also, the translation we added by hand, is still there.

  >>> user_browser.getControl(name='msgset_130_es_translation_0_new').value
  '\nYou will not erase me!'
  >>> user_browser.getControl(
  ...     name='msgset_130_es_translation_0_new_checkbox').value
  True
