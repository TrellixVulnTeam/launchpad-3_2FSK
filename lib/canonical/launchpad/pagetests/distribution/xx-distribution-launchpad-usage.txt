= Distribution Launchpad usage =

The distribution overview page indicates for what usage the 
distribution officially uses Launchpad.

    >>> anon_browser.open('http://launchpad.dev/ubuntu')
    >>> content = find_main_content(anon_browser.contents)
    >>> label_cell = content.find('th', text='Uses Launchpad for:')
    >>> print extract_text(label_cell.findNext())
    Answers
    Bug tracking
    Translations
    
The usage flags can only be set by an Launchpad administrator by 
using the 'Define Launchpad usage' action.

    >>> anon_browser.getLink('Define Launchpad usage')
    Traceback (most recent call last):
      ...
    LinkNotFound...    
    >>> anon_browser.open('http://launchpad.dev/ubuntu/+launchpad')
    Traceback (most recent call last):
      ...
    Unauthorized...
    
(Even the registrant doesn't have permission to this page.)

    >>> browser.addHeader('Authorization', 'Basic test@canonical.com:test')
    >>> browser.open('http://launchpad.dev/ubuntu/+launchpad')
    Traceback (most recent call last):
      ...
    Unauthorized...

Any Launchpad administrator can switch the checkboxes to change the official 
usage of each application:

    >>> admin_browser.open('http://launchpad.dev/ubuntu')
    >>> admin_browser.getLink('Define Launchpad usage').click()
    >>> print admin_browser.title
    Launchpad usage by Ubuntu Linux
      
    >>> print admin_browser.getControl(
    ...     'Translations for this distribution are done in Launchpad').selected
    True
    >>> print admin_browser.getControl(
    ...     'Bugs in this distribution are tracked in Launchpad').selected
    True
    >>> print admin_browser.getControl(
    ...     'People can ask questions in Launchpad Answers').selected
    True
     
Foo Bar can edit the values as well. Afterwards he gets redirected to
the distribution index page:

    >>> admin_browser.getControl(
    ...     'Translations for this distribution are done in ' +
    ...     'Launchpad').selected = False
    >>> admin_browser.getControl(
    ...     'Bugs in this distribution are tracked in Launchpad').selected \
    ...     = False
    >>> admin_browser.getControl(
    ...     'People can ask questions in Launchpad Answers').selected = False
    >>> admin_browser.getControl('Change').click()

On the distribution page, the new usage was updated.
    
    >>> content = find_main_content(admin_browser.contents)
    >>> label_cell = content.find('th', text='Uses Launchpad for:')
    >>> print extract_text(label_cell.findNext())
    &mdash;
