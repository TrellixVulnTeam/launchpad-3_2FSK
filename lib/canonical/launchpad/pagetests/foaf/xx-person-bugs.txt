= Bugs a given user is involved with =

When visiting the 'Bugs' facet of a person, the report that is displayed
lists the bug assigned to that person.

    >>> anon_browser.open('http://launchpad.dev/~name12')
    >>> anon_browser.getLink('Bugs').click()
    >>> print anon_browser.title
    Bugs related to Sample Person
    >>> print anon_browser.url
    http://bugs.launchpad.dev/~name12/+bugs

Note that we may see each bug more than once in case it's reported against
more than one target (https://launchpad.net/bugs/1357).

    >>> def print_bugs(contents):
    ...     table = find_tag_by_id(contents, 'buglisting')
    ...     for link in table.findAll('a'):
    ...         if '+bug/' in link['href']:
    ...             print link.renderContents()
    >>> print_bugs(anon_browser.contents)
    Firefox install instructions should be complete
    Reflow problems with complex page layouts
    Firefox install instructions should be complete
    Firefox does not support SVG
    Bug Title Test
    Bug Title Test
    Blackhole Trash folder
    Thunderbird crashes
    Firefox does not support SVG
    Blackhole Trash folder
    Firefox does not support SVG
    Blackhole Trash folder

This report is also the default page in the person context on the bugs
virtual host:

    >>> anon_browser.open('http://bugs.launchpad.dev/~name12')
    >>> print anon_browser.title
    Bugs related to Sample Person

== More specific listings ==

On a person +bugs page we get links to all listings of bugs we have for that
person.

=== Assigned bugs ===

    >>> anon_browser.getLink('Assigned').click()
    >>> print anon_browser.title
    Bugs assigned to Sample Person
    >>> print anon_browser.url
    http://bugs.launchpad.dev/~name12/+assignedbugs

    >>> print_bugs(anon_browser.contents)
    Firefox install instructions should be complete
    Blackhole Trash folder

=== Reported bugs ===

    >>> anon_browser.getLink('Reported').click()
    >>> print anon_browser.title
    Bugs reported by Sample Person
    >>> print anon_browser.url
    http://bugs.launchpad.dev/~name12/+reportedbugs

    >>> print_bugs(anon_browser.contents)
