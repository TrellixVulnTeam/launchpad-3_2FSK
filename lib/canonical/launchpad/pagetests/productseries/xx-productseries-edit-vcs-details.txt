= Editing the VCS details for a productseries =

Each product series is able to specify where the code is located
for that series.

If the product series has native bazaar branch then this can
be defined by selecting the Bazaar option.

    >>> browser = setupBrowser(auth='Basic test@canonical.com:test')
    >>> browser.open('http://launchpad.dev/firefox/trunk')
    >>> browser.getLink('Edit source').click()
    >>> main = find_main_content(browser.contents)
    >>> inputs = main.findAll('input')
    >>> for input in inputs:
    ...     if input['type'] == 'radio':
    ...         print '%s: %s' % (input['id'], input['value'])
    field.rcstype.0: BZR
    field.rcstype.1: CVS
    field.rcstype.2: SVN
    field.rcstype.None:

Selecting the Bazaar option allows the user to specify a branch.

    >>> browser.getControl('Bazaar').selected = True
    >>> browser.getControl(name='field.user_branch').value = (
    ...     '~name12/firefox/main')
    >>> browser.getControl('Update Details').click()
    >>> branch = find_tag_by_id(browser.contents, 'branch-details')
    >>> print extract_text(branch)
    Mozilla Firefox Mainline - Sample Person
    text
