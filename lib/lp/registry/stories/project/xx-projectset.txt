= Project Set Page =

Under "/projectgroups" there are several pages for various administration tasks.

The pages are "+review", "+new", and "+all".

A logged in, unprivileged person cannot see any of these pages.

    >>> user_browser.open('http://launchpad.dev/projectgroups/+review')
    Traceback (most recent call last):
      ...
    Unauthorized:...

    >>> user_browser.open('http://launchpad.dev/projectgroups/+all')
    Traceback (most recent call last):
      ...
    Unauthorized:...

    >>> user_browser.open('http://launchpad.dev/projectgroups/+new')
    Traceback (most recent call last):
      ...
    Unauthorized:...

An administrator may successfully access all of these pages.

    >>> admin_browser.open('http://launchpad.dev/projectgroups/+review')
    >>> admin_browser.title
    'Review Launchpad items'

    >>> admin_browser.open('http://launchpad.dev/projectgroups/+all')
    >>> admin_browser.title
    'Project groups registered in Launchpad'

    >>> admin_browser.open('http://launchpad.dev/projectgroups/+new')
    >>> admin_browser.title
    'Register a project group with Launchpad'

A member of the registry admins team my access all of these pages, too.

    >>> from zope.component import getUtility
    >>> from canonical.launchpad.interfaces.launchpad import (
    ...     ILaunchpadCelebrities)
    >>> login('foo.bar@canonical.com')
    >>> registry_member = factory.makePerson(
    ...     name='reggie', email='reggie@example.com', password='test')
    >>> celebs = getUtility(ILaunchpadCelebrities)
    >>> registry = celebs.registry_experts
    >>> registry.addMember(registry_member, registry.teamowner)
    >>> logout()

    >>> registry_browser = setupBrowser(
    ...     auth='Basic reggie@example.com:test')

    >>> registry_browser.open('http://launchpad.dev/projectgroups/+review')
    >>> registry_browser.title
    'Review Launchpad items'

    >>> registry_browser.open('http://launchpad.dev/projectgroups/+all')
    >>> registry_browser.title
    'Project groups registered in Launchpad'

    >>> registry_browser.open('http://launchpad.dev/projectgroups/+new')
    >>> registry_browser.title
    'Register a project group with Launchpad'

    registry_member.leave(registry)
