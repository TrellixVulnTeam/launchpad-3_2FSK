
  The Ubuntu distribution mirrors are shown in two different pages:

  a) the +archivemirrors page, which lists all official and enabled archive
     mirrors.

  >>> browser.open('http://launchpad.dev/distros/ubuntu/+archivemirrors')
  >>> browser.url
  'http://launchpad.dev/distros/ubuntu/+archivemirrors'

  >>> 'Archive Mirrors' in browser.contents
  True
  >>> print browser.contents
  <!DOCTYPE...
  ...Afghanistan...
  ...Archive-mirror...


  b) the +cdmirrors page, which lists all official and enabled release
     mirrors.

  >>> browser.open('http://launchpad.dev/distros/ubuntu/+cdmirrors')
  >>> browser.url
  'http://launchpad.dev/distros/ubuntu/+cdmirrors'

  >>> 'CD Mirrors' in browser.contents
  True
  >>> print browser.contents
  <!DOCTYPE...
  ...British Indian Ocean Territory...
  ...Unreachable-mirror...

  
  There's a third page, accessible only to distro owners, mirror admins of the
  distro or launchpad admins.

  >>> browser.addHeader('Authorization', 'Basic test@canonical.com:test')
  >>> browser.open('http://launchpad.dev/distros/ubuntu/+disabledmirrors')
  Traceback (most recent call last):
  ...
  Unauthorized: ...

  # XXX: This is a workaround for https://launchpad.net/bugs/45226
  >>> from zope.testbrowser.testing import Browser
  >>> browser = Browser()
  >>> browser.handleErrors = False

  >>> browser.addHeader('Authorization', 'Basic karl@canonical.com:test')
  >>> browser.open('http://launchpad.dev/distros/ubuntu/+disabledmirrors')
  >>> browser.url
  'http://launchpad.dev/distros/ubuntu/+disabledmirrors'

  >>> 'Disabled Mirrors' in browser.contents
  True
  >>> print browser.contents
  <!DOCTYPE...
  ...We don't know about any Disabled Mirrors of this distribution...
