A bug is a problem in software.

  >>> print http(r"""
  ... GET /malone/bugs/+package HTTP/1.1
  ... Authorization: Basic dGVzdEBjYW5vbmljYWwuY29tOnRlc3Q=
  ... """)
  HTTP/1.1 200 Ok
  ...
      <title>Report a bug</title>
  ...

At the moment, Malone still has a bug filing page on which you can
specify the distro in addition to the source package in which the bug
occurs.

You must be logged-in to file a bug. After you filed the bug you get
redirected to the bug page.

  >>> print http(r"""
  ... POST /malone/bugs/+package HTTP/1.1
  ... Authorization: Basic dGVzdEBjYW5vbmljYWwuY29tOnRlc3Q=
  ... Content-Length: 729
  ... Content-Type: multipart/form-data; boundary=---------------------------124775592710556272331137042681
  ... 
  ... -----------------------------124775592710556272331137042681

  ... 
  ... ubuntu
