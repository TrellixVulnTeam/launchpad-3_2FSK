There is a list of IProduct objects that have translations and are using
Rosetta officially from a IProject's translations page.

GNOME is a good example, it has products with translations.

  >>> browser.open('http://translations.launchpad.dev/gnome')
  >>> print browser.url
  http://translations.launchpad.dev/gnome

Evolution being one with translations and being used officially

  >>> browser.getLink('Evolution').click()
  >>> print browser.url
  http://translations.launchpad.dev/evolution

  # We should use goBack() here but can't because of ZC #645:
  # zope.testbrowser truncates document content after goBack().
  #>>> browser.goBack()
  >>> browser.open('http://translations.launchpad.dev/gnome')
  >>> print browser.url
  http://translations.launchpad.dev/gnome

Netapplet is another product of GNOME project. It has translations, but it's not
using Rosetta officially, so it's not in this page.

  >>> browser.getLink('Network Applet')
  Traceback (most recent call last):
  ...
  LinkNotFoundError

Let's confirm what we just stated.

  >>> browser.open('http://launchpad.dev/gnome')
  >>> print browser.url
  http://launchpad.dev/gnome

Netapplet is in fact a product belonging to GNOME project.

  >>> netapplet_link = browser.getLink('Network Applet')
  >>> netapplet_link.click()
  >>> print browser.url
  http://launchpad.dev/netapplet

It's not using Rosetta officially

  >>> print find_portlet(browser.contents, 'Network Applet')
  <...
  ...Product status:...
  ...Doesn&rsquo;t use Rosetta...

And it has translations.

  >>> browser.open('http://translations.launchpad.dev/netapplet')
  >>> print browser.url
  http://translations.launchpad.dev/netapplet

  >>> netapplet_template = browser.getLink('View Template & All Languages...')
  >>> print netapplet_template.url
  http://translations.launchpad.dev/netapplet/trunk/+pots/netapplet

The translations page has also a link to the product search page to see
whether a product is registered in Launchpad.

  >>> browser.open('http://translations.launchpad.dev/gnome')
  >>> print browser.url
  http://translations.launchpad.dev/gnome
  >>> browser.getLink('registered with Launchpad').click()
  >>> print browser.url
  http://translations.launchpad.dev/products
