= Subscribers portlet content view =

The view for the subscribers portlet exposes methods for getting the
subscription data ready for display. The list of subscribers is reordered so
that subscriptions that the current viewer can unsubscriber appear at the top
of the list.

    >>> from lp.bugs.browser.bugsubscription import (
    ...     BugPortletSubcribersContents)
    >>> login('foo.bar@canonical.com')
    >>> reporter = factory.makePerson(displayname='Bug Reporter')
    >>> bug = factory.makeBug(owner=reporter)
    >>> view = BugPortletSubcribersContents(bug, None)
    >>> bug.subscribe(view.user, view.user)
    <BugSubscription at ...>
    >>> bug.subscribe(view.user.teams_participated_in[0], view.user)
    <BugSubscription at ...>
    >>> for subscription in view.sorted_direct_subscriptions:
    ...     print '%s   %s' % (
    ...         subscription.person.displayname,
    ...         subscription.canBeUnsubscribedByUser(view.user))
    Foo Bar                        True
    Canonical Partner Developers   True
    Bug Reporter                   False
