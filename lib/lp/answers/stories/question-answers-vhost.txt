= Answers Default Pages =

The answers.launchpad.net domain is used for the Answer Tracker
application. When visiting objects on that virtual host, the default
selected facet is the Answers facet.

== Product ==

    >>> anon_browser.open('http://answers.launchpad.dev/firefox')
    >>> print anon_browser.title
    Questions for Mozilla Firefox : Mozilla Firefox

== Distribution ==

    >>> anon_browser.open('http://answers.launchpad.dev/ubuntu')
    >>> print anon_browser.title
    Questions for Ubuntu Linux : Ubuntu

== Soure Package ==

    >>> anon_browser.open(
    ...     'http://answers.launchpad.dev/ubuntu/+source/mozilla-firefox')
    >>> print anon_browser.title
    Questions for “mozilla-firefox” package in Ubuntu : “mozilla-firefox” package : Ubuntu

    >>> anon_browser.open(
    ...     'http://answers.launchpad.dev/ubuntu/hoary/+source/mozilla-firefox')
    >>> print anon_browser.title
    Questions for mozilla-firefox in Ubuntu Hoary : “mozilla-firefox” package : 5.04 : Ubuntu

== Project ==

    >>> anon_browser.open(
    ...     'http://answers.launchpad.dev/mozilla')
    >>> print anon_browser.title
    Questions for The Mozilla Project : the Mozilla Project

== Person ==

    >>> anon_browser.open('http://answers.launchpad.dev/~name16')
    >>> print anon_browser.title
    Questions involving Foo Bar : Foo Bar
