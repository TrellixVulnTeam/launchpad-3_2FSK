There is a list of IProduct objects that have translations and are using
Rosetta officially from a IProject's translations page.

GNOME is a good example, it has products with translations.

  >>> browser.open('http://launchpad.dev/gnome/+translations')
  >>> print browser.url
  http://launchpad.dev/gnome/+translations

Evolution being one with translations and being used officially

  >>> evo_link = browser.getLink('The Evolution Groupware Application')
  >>> print evo_link.url
  http://launchpad.dev/evolution/+translations

Netapplet is another product of GNOME project. It has translations, but it's not
using Rosetta officially, so it's not in this page.

  >>> netapplet_link = browser.getLink('Network Applet')
  Traceback (most recent call last):
  ...
  LinkNotFoundError

Let's confirm what we just stated.

  >>> browser.open('http://launchpad.dev/gnome')
  >>> print browser.url
  http://launchpad.dev/gnome

Netapplet is in fact a product belonging to GNOME project.

  >>> netapplet_link = browser.getLink('Network Applet')
  >>> netapplet_link.click()
  >>> print browser.url
  http://launchpad.dev/netapplet

It's not using Rosetta officially

  >>> u'Doesn&rsquo;t use Rosetta' in browser.contents
  True

And it has translations.

  >>> browser.open('http://launchpad.dev/netapplet/+translations')
  >>> print browser.url
  http://launchpad.dev/netapplet/+translations

  >>> netapplet_template = browser.getLink('View Template & All Languages...')
  >>> print netapplet_template.url
  http://launchpad.dev/netapplet/trunk/+pots/netapplet
