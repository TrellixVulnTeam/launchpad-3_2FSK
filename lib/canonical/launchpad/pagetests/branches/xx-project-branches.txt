= Project Branches Overview =

Projects link their branch listing page.

   >>> browser.open('http://launchpad.dev/mozilla')
   >>> browser.getLink('Code').click()
   >>> print browser.title
   The Mozilla Project's Bazaar branches registered in Launchpad
  
== Default page for code site ==

When going directly to a project on the code rootsite for launchpad,
the branch listing is the default page shown.

If there are branches, then they are displayed.

   >>> browser.open('http://code.launchpad.dev/mozilla')
   >>> print browser.title
   The Mozilla Project's Bazaar branches registered in Launchpad
   >>> table = find_tag_by_id(browser.contents, 'branchtable')
   >>> for row in table.tbody.fetch('tr'):
   ...     print extract_text(row)
    main            Sample Person  5 New                                      firefox
    release--0.9.1                 5 New  2005-03-09 15:45:00  ... weeks ago  firefox
    release-0.8                    5 New                                      firefox
    release-0.9                    5 New  2005-03-09 15:50:00  ... weeks ago  firefox
    release-0.9.2                  5 New  2005-03-09 15:40:00  ... weeks ago  firefox
    main            Mark Shuttleworth  5 New                              thunderbird

If there are not any branches, a relevant message is shown.

   >>> browser.open('http://code.launchpad.dev/aaa')
   >>> print browser.title
   The Test Project's Bazaar branches registered in Launchpad
   >>> print find_tag_by_id(browser.contents, 'no-branch-message').renderContents()
   There may be branches registered for the Test Project but none of
   them match the current filter criteria for this page.
   Try filtering on "Any Status".

OK, lets follow the advice.

   >>> browser.getControl(name='field.lifecycle').displayValue = ['Any Status']
   >>> browser.getControl('Filter').click()
   >>> print find_tag_by_id(browser.contents, 'no-branch-message').renderContents()
   There are no branches registered for the Test Project in Launchpad today.
   We recommend you visit
   <a href="http://www.bazaar-vcs.org">www.bazaar-vcs.org</a>
   for more information about how you can use
   the Bazaar revision control system to improve community
   participation in this project group.
