Let's see if we can load the distro's add page:

  >>> from zope.testbrowser import Browser
  >>> browser = Browser()
  >>> browser.addHeader('Authorization', 'Basic foo.bar@canonical.com:test')

  >>> browser.open("http://localhost:9000/distros/+add")
  >>> browser.contents
  '...<h1>Register a distribution in Soyuz</h1>...'

And now let's create a Test Distro:

  >>> browser.getControl(name="field.name").value = 'test'
  >>> browser.getControl("Display Name").value = 'Test Distro'
  >>> browser.getControl("Title").value = 'The Test Distribution'
  >>> browser.getControl("Summary").value = 'Test Distro Summary'
  >>> browser.getControl("Description").value = 'Test Distro Description'
  >>> browser.getControl("Domain name").value = 'foo.com'
  >>> browser.getControl("Members").value = 'sabdfl'
  >>> browser.getControl("Add").click()
  >>> browser.url
  'http://localhost:9000/distros/test'
  >>> browser.contents
  '...The Test Distribution...'
