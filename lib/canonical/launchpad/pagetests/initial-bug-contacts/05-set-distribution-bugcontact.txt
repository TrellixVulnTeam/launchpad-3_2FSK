On the page for a distribution, there is a link for changing the bug contact.

  >>> distropage = http(r"""
  ... GET /distros/ubuntu/+bugs HTTP/1.1
  ... Authorization: Basic bWFya0BoYmQuY29tOnRlc3Q=
  ... """).getBody()
  >>> "Change Bug Contact" in distropage
  True

This link is not available if you are not logged in with permission to change
the bug contact.

  >>> distropage = http(r"""
  ... GET /distros/ubuntu/+bugs HTTP/1.1
  ... """).getBody()
  >>> "Change Bug Contact" in distropage
  False

Anyone with launchpad.Edit permission can edit the distribution bug
contact. In this example, we'll use mark@hbd.com to set the distribution
bug contact.

  >>> print http(r"""
  ... GET /distros/ubuntu/+bugcontact HTTP/1.1
  ... Authorization: Basic bWFya0BoYmQuY29tOnRlc3Q=
  ... """)
  HTTP/1.1 200 Ok
  ...

The bug contact page takes just one simple value: the bug contact
email or nickname. Let's set carlos@canonical.com as the bug contact for
Ubuntu.

  >>> print http(r"""
  ... POST /distros/ubuntu/+bugcontact HTTP/1.1
  ... Authorization: Basic bWFya0BoYmQuY29tOnRlc3Q=
  ... Content-Length: 234
  ... Content-Type: multipart/form-data; boundary=----------0xKhTmLbOuNdArY
  ... Referer: http://192.168.1.3:9000/distros/ubuntu/+bugcontact
  ... 
  ... ------------0xKhTmLbOuNdArY
  ... Content-Disposition: form-data; name="field.bugcontact"
  ... 
  ... carlos@canonical.com
  ... ------------0xKhTmLbOuNdArY
  ... Content-Disposition: form-data; name="UPDATE_SUBMIT"
  ... 
  ... Change
  ... ------------0xKhTmLbOuNdArY--
  ... """)
  HTTP/1.1 303 See Other
  ...
  Location: .../distros/ubuntu
  ...
