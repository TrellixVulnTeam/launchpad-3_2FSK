To set a bug contact for a distribution, we need to go to the Bug page of that
distribution:

  >>> browser.open('http://localhost:9000/distros/ubuntu/+bugs')

But the link is not available if you are not logged in with permission to change
the bug contact.

  >>> browser.getLink("Change Bug Contact")
  Traceback (most recent call last):
  ...
  LinkNotFoundError

Now we login as Mark... 

  >>> browser.addHeader('Authorization', 'Basic mark@hbd.com:test')
  >>> browser.reload()
  >>> browser.url
  'http://localhost:9000/distros/ubuntu/+bugs'

...and we can see that the link is there.

  >>> bugcontact_link = browser.getLink("Change Bug Contact")
  >>> bugcontact_link.url
  'http://localhost:9000/distros/ubuntu/+bugcontact'


Anyone with launchpad.Edit permission can edit the distribution bug
contact. In this example, we're using mark@hbd.com to set the distribution
bug contact.

  >>> bugcontact_link.click()
  >>> browser.url
  'http://localhost:9000/distros/ubuntu/+bugcontact'

The bug contact page takes just one simple value: the bug contact
email or nickname. Let's set carlos@canonical.com as the bug contact for
Ubuntu.

  >>> browser.getControl("Bug Contact").value = 'carlos@canonical.com'
  >>> browser.getControl("Change").click()

And then we're redirected to the distribution page

  >>> browser.url
  'http://localhost:9000/distros/ubuntu'

  >>> message = (
  ...   'class="informational message">Successfully changed the distribution '
  ...   'bug contact to carlos@canonical.com')
  >>> message in browser.contents
  True

Now that the bug contact is set, there's a link to change it at the
distribution page.

  >>> browser.getLink(
  ...   url='http://localhost:9000/distros/ubuntu/+bugcontact').url
  'http://localhost:9000/distros/ubuntu/+bugcontact'
