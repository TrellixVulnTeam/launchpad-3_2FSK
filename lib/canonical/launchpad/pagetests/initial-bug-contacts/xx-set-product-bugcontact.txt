= Teams as Bug Contacts =

First, some setup. We create a new product owned by "Sample Person".

    >>> browser.addHeader("Authorization", 
    ...                   "Basic test@canonical.com:test")
    >>> browser.open('http://launchpad.dev/projects/+new')
    >>> browser.getControl(name='field.name').value = 'testy'
    >>> browser.getControl('Display Name:').value = 'Test Product'
    >>> browser.getControl('Title:').value = 'A product for testing things'
    >>> browser.getControl('Summary:').value = 'Testing team bug contacts'
    >>> browser.getControl('Description:').value = 'See bug 109652'
    >>> browser.getControl('Add').click()
    >>> browser.url
    'http://launchpad.dev/testy'

Sample Person is also a member of "Warty Security Team". He however should 
not be able to set Warty Security Team as a bug contact for Test Project as 
he's not an admin of that team.

    >>> browser.getLink('Bugs').click()
    >>> browser.getLink('Change bug contact').click()
    >>> browser.url
    'http://bugs.launchpad.dev/testy/+bugcontact'

    >>> browser.getControl('Bug Contact').value = 'name20'
    >>> browser.getControl('Change').click()
    >>> 'Successfully changed the bug contact' not in browser.contents
    True

    >>> print browser.contents
    <!DOCTYPE...
    ...You cannot set Warty Security Team as the bug contact...
    ...you are not an administrator of that team...

In the warning, the user will be pointed to the Warty Security Team's
membership page and advised to contact an administrator if she believes that 
the team should be a bug contact.

    >>> browser.getLink('Warty Security Team administrators').click()
    >>> browser.url
    'http://launchpad.dev/~name20/+members'
