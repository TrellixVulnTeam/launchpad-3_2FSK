Please edit d2x.ini and do the following:

* change '-hires_textures 1' to '-hires_textures 0'
* change '-hires_models 1' to '-hires_models 0'
* change '-gl_alttexmerge 1' to '-gl_alttexmerge 0'

You can also try to change ';-max_segments 900' to '-max_segments 900'
(just remove the ';') - that will limit the max. level size you can play to
the max. level size of standard Descent 2 (D2X-XL allows up to 6000
segments per level).

From the Windows distribution rar file:

* extract d2x-xl-win16.exe, sdl-win16.dll and sdl_mixer-win16.dll and
rename them to d2x-xl.exe, sdl.dll and sdl_mixer.dll

D2X-XL has not been built for Win9X/ME, so if that doesn't help, there is
nothing else I could do for you.

You might try and increase your machine's main memory (if it has very
little RAM) and install a graphics card with more video memory (a Geforce
FX 5200 with 128 MB would be good enough).
