IPOTemplate objects have IProductSeries or ISourcePackage as its direct
structural object.

Taking an IPOTemplate for IProductSeries context:

  >>> browser.open(
  ...     'http://translations.launchpad.dev/evolution/trunk/+pots/'
  ...     'evolution-2.2')
  >>> print browser.url
  http://translations.launchpad.dev/evolution/trunk/+pots/evolution-2.2

The Application tabs should point to IProductSeries URLs.

  >>> print_tab_links(browser.contents)
  Overview: http://launchpad.dev/evolution/trunk/
  Code: Unavailable
  Bugs: http://bugs.launchpad.dev/evolution/trunk/
  Blueprints: http://blueprints.launchpad.dev/evolution/trunk/
  Translations: http://translations.launchpad.dev/evolution/trunk/
  Answers: Unavailable

But the actions should point to IPOTemplate urls.

  >>> print_action_links(browser.contents)
  Show translation status: http://translations.launchpad.dev/evolution/trunk/+pots/evolution-2.2/
  Upload a file: http://translations.launchpad.dev/evolution/trunk/+pots/evolution-2.2/+upload
  Download translations: http://translations.launchpad.dev/evolution/trunk/+pots/evolution-2.2/+export
  Change details: http://translations.launchpad.dev/evolution/trunk/+pots/evolution-2.2/+edit


Taking an IPOTemplate for ISourcePackage context:

  >>> browser.open(
  ...     'http://translations.launchpad.dev/ubuntu/hoary/+source/evolution/'
  ...     '+pots/evolution-2.2')
  >>> print browser.url
  http://translations.launchpad.dev/ubuntu/hoary/+source/evolution/+pots/evolution-2.2

The Application tabs should point to ISourcePackage URLs.

  >>> print_tab_links(browser.contents)
  Overview: http://launchpad.dev/ubuntu/hoary/+source/evolution/
  Code: Unavailable
  Bugs: http://bugs.launchpad.dev/ubuntu/hoary/+source/evolution/
  Blueprints: Unavailable
  Translations: http://translations.launchpad.dev/ubuntu/hoary/+source/evolution/
  Answers: http://answers.launchpad.dev/ubuntu/hoary/+source/evolution/

But the actions should point to IPOTemplate urls.

  >>> print_action_links(browser.contents)
  Show translation status: http://translations.launchpad.dev/ubuntu/hoary/+source/evolution/+pots/evolution-2.2/
  Upload a file: http://translations.launchpad.dev/ubuntu/hoary/+source/evolution/+pots/evolution-2.2/+upload
  Download translations: http://translations.launchpad.dev/ubuntu/hoary/+source/evolution/+pots/evolution-2.2/+export
  Change details: http://translations.launchpad.dev/ubuntu/hoary/+source/evolution/+pots/evolution-2.2/+edit
