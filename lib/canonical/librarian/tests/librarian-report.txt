We have a report that we run as necessary to see what is using the Librarian
storage.

    >>> import os.path
    >>> from canonical.config import config
    >>> path = os.path.join(config.root, 'scripts', 'librarian-report.py')

    >>> import subprocess
    >>> proc = subprocess.Popen(
    ...     path, stdout=subprocess.PIPE, stderr=subprocess.STDOUT,
    ...     stdin=subprocess.PIPE)
    >>> (out, ignored) = proc.communicate()
    >>> print '\n' + out
    <BLANKLINE>
    ...
    133 bytes    hwsubmission in 2 files
    ...


We can filter on date to produce deltas.

    >>> proc = subprocess.Popen(
    ...     [path, '--from=2005/01/01', '--until=2005/12/31'],
    ...     stdout=subprocess.PIPE, stderr=subprocess.STDOUT,
    ...     stdin=subprocess.PIPE)
    >>> (out, ignored) = proc.communicate()
    >>> print '\n' + out
    <BLANKLINE>
    ...
    0 bytes      hwsubmission in 0 files
    ...
