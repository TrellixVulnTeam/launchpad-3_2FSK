= Products with code =

The main purpose of this page is to show to casual browsers (and robots) that
there are branches associated with products, and to entice them to click
through to get lists of branches of products that interest them.

    >>> browser.open('http://code.launchpad.dev')
    >>> browser.getLink('branches registered in').click()
    >>> print browser.title
    Projects with Code

You can also get to this page as the default view on the code rootsite
for the IProductSet.

    >>> browser.open('http://code.launchpad.dev/products')
    >>> print browser.title
    Products with Code

    >>> contents = find_tag_by_id(browser.contents, 'maincontent')
    >>> for anchor in contents.fetch('a'):
    ...     print anchor['href']
    http://code.launchpad.dev/evolution
    http://code.launchpad.dev/firefox
    http://code.launchpad.dev/gnome-terminal
    http://code.launchpad.dev/iso-codes
    http://code.launchpad.dev/thunderbird
