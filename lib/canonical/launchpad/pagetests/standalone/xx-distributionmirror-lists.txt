
  The Ubuntu distribution mirrors are shown in two different pages:

  a) the +archivemirrors page, which lists all official and enabled archive
     mirrors.

  >>> browser.open('http://launchpad.dev/ubuntu/+archivemirrors')
  >>> browser.url
  'http://launchpad.dev/ubuntu/+archivemirrors'

  >>> from BeautifulSoup import BeautifulSoup
  >>> print find_main_content(browser.contents).find('h1').renderContents()
  Official Archive Mirrors
  >>> for tr in find_tag_by_id(browser.contents, 'mirrors_list').findAll('tr'):
  ...     if tr.find('th'):
  ...         print extract_text(tr.find('th'))
  ...     else:
  ...         print extract_text(tr.find('td'))
  <BLANKLINE>
  Antarctica...
  Archive-mirror2...
  France...
  Archive-404-mirror...
  Archive-mirror...
  United Kingdom...
  Canonical-archive...

  b) the +cdmirrors page, which lists all official and enabled release
     mirrors.

  >>> browser.open('http://launchpad.dev/ubuntu/+cdmirrors')
  >>> browser.url
  'http://launchpad.dev/ubuntu/+cdmirrors'

  >>> print find_main_content(browser.contents).find('h1').renderContents()
  Official CD Mirrors
  >>> for tr in find_tag_by_id(browser.contents, 'mirrors_list').findAll('tr'):
  ...     if tr.find('th'):
  ...         print extract_text(tr.find('th'))
  ...     else:
  ...         print extract_text(tr.find('td'))
  <BLANKLINE>
  France...
  Releases-mirror...
  Unreachable-mirror...
  Germany...
  Releases-mirror2...
  United Kingdom...
  Canonical-releases...

  
  There's a third page, accessible only to distro owners, mirror admins of the
  distro or launchpad admins.

  >>> browser.addHeader('Authorization', 'Basic test@canonical.com:test')
  >>> browser.open('http://launchpad.dev/ubuntu/+disabledmirrors')
