The bug contact reports present an overview and searchable bug listing
views for packages for which a user is a bug contact.

  >>> print http(r"""
  ... GET /people/name16/+packagebugs HTTP/1.1
  ... """)
  HTTP/1.1 200 Ok
  ...
  ...Foo Bar's package bug reports...
  ...
  ...mozilla-firefox in ubuntu...
  ...pmount in ubuntu...
  ...

The overview report presents links to open, critical, unassigned, and in
progress bugs.

  >>> print http(r"""
  ... GET /people/name16/+packagebugs-search?field.distribution=ubuntu&field.sourcepackagename=mozilla-firefox&field.status=Unconfirmed&field.status=Confirmed&field.status=In+Progress&search=Search HTTP/1.1
  ... """)
  HTTP/1.1 200 Ok
  ...

  >>> print http(r"""
  ... GET /people/name16/+packagebugs-search?field.distribution=ubuntu&field.importance=Critical&field.sourcepackagename=mozilla-firefox&field.status=Unconfirmed&field.status=Confirmed&field.status=In+Progress&search=Search HTTP/1.1
  ... """)
  HTTP/1.1 200 Ok
  ...

  >>> print http(r"""
  ... GET /people/name16/+packagebugs-search?field.distribution=ubuntu&field.sourcepackagename=mozilla-firefox&field.status=Unconfirmed&field.status=Confirmed&field.status=In+Progress&field.unassigned=on&search=Search HTTP/1.1
  ... """)
  HTTP/1.1 200 Ok
  ...

  >>> print http(r"""
  ... GET /people/name16/+packagebugs-search?field.distribution=ubuntu&field.sourcepackagename=mozilla-firefox&field.status=In+Progress&search=Search HTTP/1.1
  ... """)
  HTTP/1.1 200 Ok
  ...

  >>> print http(r"""
  ... GET /people/name16/+packagebugs-search?field.distribution=debian&field.sourcepackagename=mozilla-firefox&field.status=Unconfirmed&field.status=Confirmed&field.status=In+Progress&search=Search HTTP/1.1
  ... """)
  HTTP/1.1 200 Ok
  ...

On the bug listing page, there is a searchtext field, for searching the
title and description of the package's bugs.

  >>> print http(r"""
  ... GET /people/name16/+packagebugs-search?field.distribution=debian&field.sourcepackagename=mozilla-firefox&field.searchtext=support&search=Search HTTP/1.1
  ... """)
  HTTP/1.1 200 Ok
  ...
