This page shows a list of PO templates contained within all source packages
for the distrorelease that is the translation target for a given distribution.

  >>> browser.open('http://localhost/distros/ubuntu/+translations')
  >>> browser.url
  'http://localhost/distros/ubuntu/+translations'

Check that there aren't disabled languages shown here:

  >>> 'Spanish (Spain)' not in browser.contents
  True
  >>> 'Italian (Italy)' not in browser.contents
  True

But we show the other languages

  >>> 'Spanish' in browser.contents
  True
  >>> 'Italian' in browser.contents
  True

Portuguese (Brazil) is a country specific language but it should not be
disabled and appear in our list.

  >>> 'Portuguese (Brazil)' in browser.contents
  True


Now, we are going to check that the language list we got is pointing to the
right translation target.

  >>> browser.getLink('Spanish')
  <Link text='Spanish' url='http://localhost/distros/ubuntu/hoary/+lang/es'>
  >>> browser.getLink('Italian')
  <Link text='Italian' url='http://localhost/distros/ubuntu/hoary/+lang/it'>
  >>> browser.getLink('Portuguese (Brazil)')
  <Link text='Portuguese (Brazil)' url='http://localhost/distros/ubuntu/hoary/+lang/pt_BR'>

And the other Ubuntu distributions should be there too.

  >>> 'Other releases of Ubuntu' in browser.contents
  True
  >>> 'Breezy Badger Autotest' in browser.contents
  True
  >>> 'The Grumpy Groundhog Release' in browser.contents
  True
  >>> 'The Warty Warthog Release' in browser.contents
  True

We are not showing its translation status here, so we should have links to
their particular translation status.

  >>> browser.getLink('6.6.6')
  <Link text='6.6.6' url='http://localhost/distros/ubuntu/breezy-autotest/+translations'>
  >>> browser.getLink('5.10')
  <Link text='5.10' url='http://localhost/distros/ubuntu/grumpy/+translations'>
  >>> browser.getLink('4.10')
  <Link text='4.10' url='http://localhost/distros/ubuntu/warty/+translations'>

But we are already showing the status for the translation target one,
we should not have a link to it and thus, we get the link to a more general
link from the portlets.

  >>> browser.getLink('5.04')
  <Link text='hoary 5.04 (DEVELOPMENT)' url='http://localhost/distros/ubuntu/hoary'>

Let's try now a distribution that lacks a translation target.
Debian is a good example.

  >>> browser.open('http://localhost/distros/debian/+translations')
  >>> browser.url
  'http://localhost/distros/debian/+translations'
  >>> 'This distribution has a prefered release to be translated' in browser.contents
  False
  >>> 'Other releases of' in browser.contents
  False

But we still show the list of distrorelease to reach their translations.

  >>> browser.getLink('3.0')
  <Link text='3.0' url='http://localhost/distros/debian/woody/+translations'>
  >>> browser.getLink('3.1')
  <Link text='3.1' url='http://localhost/distros/debian/sarge/+translations'>
  >>> browser.getLink('3.2')
  <Link text='3.2' url='http://localhost/distros/debian/sid/+translations'>
