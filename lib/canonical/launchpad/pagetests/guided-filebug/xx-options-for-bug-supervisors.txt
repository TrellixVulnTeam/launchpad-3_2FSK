= Bug filing options for bug supervisors =

During the bug filing process, normal or advanced, bug supervisors are
given extra options.

    >>> from canonical.launchpad.ftests.bug import print_bug_affects_table


== Setting the importance ==

Bug supervisors can set the importance of bugs as they are being
filed.

    >>> user_browser.open(
    ...     'http://bugs.launchpad.dev/firefox/+filebug')
    >>> user_browser.getControl('Summary').value = "Bug with tags"
    >>> user_browser.getControl('Continue').click()
    >>> user_browser.getControl('Importance').displayValue
    ['Undecided']
    >>> user_browser.getControl('Importance').displayValue = ['High']
    >>> user_browser.getControl('Further information').value = "Blah"
    >>> user_browser.getControl('Submit Bug Report').click()
    >>> print_bug_affects_table(user_browser.contents)
    Mozilla Firefox
    New
    High
