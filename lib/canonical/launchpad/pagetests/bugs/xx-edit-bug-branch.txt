= Edit a BugBranch =

When a branch is associated with a bug it's possible to edit its status,
which tells the progress on the fix.

The bug branches are displayed on the bug page with their status
clickable.

    >>> user_browser.open('http://launchpad.dev/bugs/4')
    >>> print extract_text(
    ...     find_tag_by_id(user_browser.contents, 'bug_branch_details'))
    Related branch            Developer          Status for this bug
    Mozilla Firefox 0.9.2     Mark Shuttleworth  Abandoned Attempt
    Mozilla Firefox Mainline  Sample Person      Fix In Progress
    >>> user_browser.getLink('Fix In Progress').click()

Now we can edit the status.

    >>> user_browser.getControl('Fix Status').displayValue = [
    ...     'Best Fix Available']
    >>> user_browser.getControl('Whiteboard').value = 'A hack, but it works.'
    >>> user_browser.getControl('Save Changes').click()
    >>> print extract_text(
    ...     find_tag_by_id(user_browser.contents, 'bug_branch_details'))
    Related branch            Developer          Status for this bug
    Mozilla Firefox 0.9.2     Mark Shuttleworth  Abandoned Attempt
    Mozilla Firefox Mainline  Sample Person      Best Fix Available
