
  If you're not logged in and go to the +polls page of the "Ubuntu Team"
  you'll see a link to login as a team administrator.

  >>> print http(r"""
  ... GET /people/name17/+polls HTTP/1.1
  ... """)
  HTTP/1.1 200 Ok
  ...
        <a href="+login">Log in as an admin to set up a new poll...
  ...


  Try to create a new poll logged in as 'no-priv', which is not a team
  administrator. 

  >>> print http(r"""
  ... GET /people/name17/+polls/+new HTTP/1.1
  ... Authorization: Basic bm8tcHJpdkBjYW5vbmljYWwuY29tOnRlc3Q=
  ... """)
  HTTP/1.1 403 Forbidden
  ...
          Sorry, you don't have permission to access this page.
  ...


  Now we're logged in as 'name16' which is a team administrator and thus can
  create a new poll.

  >>> print http(r"""
  ... GET /people/name17/+polls HTTP/1.1
  ... Authorization: Basic Zm9vLmJhckBjYW5vbmljYWwuY29tOnRlc3Q=
  ... """)
  HTTP/1.1 200 Ok
  ...
      <ul class="add">
        <li><a href="+new">Set up a new poll</a></li>
  ...
