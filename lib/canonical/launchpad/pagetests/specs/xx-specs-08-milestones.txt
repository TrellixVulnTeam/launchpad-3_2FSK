Specifications and Milestones
=============================

In terms of project management, lets go ahead and target one of these specs
to the 1.0 milestone of Mozilla. We'll use the "canvas" spec. First we
ensure we can see the page for milestone targeting.

  >>> admin_browser.open(
  ...     'http://blueprints.launchpad.dev/firefox/+spec/canvas')
  >>> admin_browser.getLink('Target milestone').click()
  >>> admin_browser.title
  'Target feature to milestone'

Now, we choose a milestone from the list.

  >>> admin_browser.getControl('Milestone').displayOptions
  ['(no value)', 'Mozilla Firefox: 1.0']
  >>> admin_browser.getControl('Milestone').value = ['1']
  >>> admin_browser.getControl('Status Whiteboard').value = 'foo'
  >>> admin_browser.getControl('Change').click()

We expect to be redirected to the spec home page.

  >>> admin_browser.url
  'http://blueprints.launchpad.dev/firefox/+spec/canvas'

And on that page, we expect to see that the spec is targeted to the 1.0
milestone.

  >>> find_main_content(admin_browser.contents)
  <...Milestone target:
  <...http://blueprints.launchpad.dev/firefox/+milestone/1.0...

  >>> admin_browser.getLink('1.0').url
  'http://blueprints.launchpad.dev/firefox/+milestone/1.0'
  >>> admin_browser.getLink('1.0').click()

Now, we should see the spec on the Mozilla Milestone 1.0 page.

  >>> admin_browser.getLink('Support <canvas> Objects').url
  'http://blueprints.launchpad.dev/firefox/+spec/canvas'
