Malone supports monitoring the status of a bug in an external bug
tracker. To monitor the status of a bug in an external bug tracker,
that bug tracker must be registered.

You can find out which bug trackers are currently registered by
visited the Known Bug Tracking Systems page.

  >>> print http(r"""
  ... GET /malone/bugtrackers HTTP/1.1
  ... Referer: http://localhost:9000/malone/bugtrackers/newbugtracker
  ... """)
  HTTP/1.1 200 Ok
  Content-Length: ...
  Content-Type: text/html;charset=utf-8
  <BLANKLINE>
  ...
    <h1>Known Bug Tracking Systems</h1>
  ...

As user Foo Bar, let's click on the Register Your Bug Tracker link to
register a new bug tracker.

  >>> print http(r"""
  ... GET /malone/bugtrackers/+newbugtracker HTTP/1.1
  ... Authorization: Basic Zm9vLmJhckBjYW5vbmljYWwuY29tOnRlc3Q=
  ... Referer: http://localhost:9000/malone/bugtrackers
  ... """)
  HTTP/1.1 200 Ok
  Content-Length: ...
  Content-Type: text/html;charset=utf-8
  <BLANKLINE>
  ...
      <h1>Register a New Bug Tracking System</h1>
  ...
                  <input type="text" name="name" size="15" value="" style="text-transform: lowercase" />
  ...

Fill in all the necessary info about this bug tracker...

  >>> print http(r"""
  ... POST /malone/bugtrackers/+newbugtracker HTTP/1.1
  ... Authorization: Basic Zm9vLmJhckBjYW5vbmljYWwuY29tOnRlc3Q=
  ... Content-Length: 152
  ... Content-Type: application/x-www-form-urlencoded
  ... Referer: http://localhost:9000/malone/bugtrackers/+newbugtracker
  ... 
  ... name=testbugtrck&title=Test+bug+Tracker&summary=Test+test&baseurl=http%3A%2F%2Ftest.bugs.com%2F&contactdetails=sdfdsfsdfdf&Register=Register+Bug+Tracker""")
  HTTP/1.1 303 See Other
  Content-Length: ...
  Content-Type: text/html;charset=utf-8
  Location: http://localhost:9000/malone/bugtrackers
  <BLANKLINE>
  ...

And now, we can see that the new bug tracker is registered.

  >>> print http(r"""
  ... GET /malone/bugtrackers HTTP/1.1
  ... Referer: http://localhost:9000/malone/bugtrackers/+newbugtracker
  ... """)
  HTTP/1.1 200 Ok
  Content-Length: ...
  Content-Type: text/html;charset=utf-8
  <BLANKLINE>
  ...
  ...Test bug Tracker...
  ...

Click on the bug tracker's row in the table listing to view the info
page for that bug tracker.

  >>> print http(r"""
  ... GET /malone/bugtrackers/testbugtrck HTTP/1.1
  ... Authorization: Basic Zm9vLmJhckBjYW5vbmljYWwuY29tOnRlc3Q=
  ... Referer: http://localhost:9000/malone/bugtrackers
  ... """)
  HTTP/1.1 200 Ok
  Content-Length: ...
  Content-Type: text/html;charset=utf-8
  <BLANKLINE>
  ...
      <h1>Test bug Tracker</h1>
  ...

Any logged-in user can view the edit page of the bug tracker.

  >>> print http(r"""
  ... GET /malone/bugtrackers/testbugtrck/+edit HTTP/1.1
  ... Authorization: Basic Zm9vLmJhckBjYW5vbmljYWwuY29tOnRlc3Q=
  ... Referer: http://localhost:9000/malone/bugtrackers/testbugtrck
  ... """)
  HTTP/1.1 200 Ok
  Content-Length: ...
  Content-Type: text/html;charset=utf-8
  <BLANKLINE>
  ...
                  <input type="text" name="title" size="30"
                         value="Test bug Tracker" />
  ...

Modifications to the bug tracker can be made as any logged in user:

  >>> print http(r"""
  ... POST /malone/bugtrackers/testbugtrck/+edit HTTP/1.1
  ... Authorization: Basic Zm9vLmJhckBjYW5vbmljYWwuY29tOnRlc3Q=
  ... Content-Length: 131
  ... Content-Type: application/x-www-form-urlencoded
  ... Referer: http://localhost:9000/malone/bugtrackers/testbugtrck/+edit
  ... 
  ... title=Test+Bug+Tracker&summary=Test+test&baseurl=http%3A%2F%2Ftest.bugs.com%2F&contactdetails=sdfdsfsdfdf&Update=Update+Bug+Tracker""")
  HTTP/1.1 303 See Other
  Content-Length: ...
  Content-Type: text/html;charset=utf-8
  Location: http://localhost:9000/malone/bugtrackers/testbugtrck
  <BLANKLINE>
  ...
