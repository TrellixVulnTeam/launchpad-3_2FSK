= Branch Details on Product Series =

== No links ==

If the product series has no associated branch then the 
revision control portlet should show 'None given'

  >>> user_browser.open('http://launchpad.dev/firefox/trunk')
  >>> print find_portlet(user_browser.contents, 'Revision control details')
  <div class="portlet">
  ...<i>None given</i>...
  </div>

  
== Add association ==

Now if we associate a branch it should be there

  >>> user_browser.getLink('Change Series Details').click()
  >>> user_browser.url
  'http://launchpad.dev/firefox/trunk/+edit'
  >>> user_browser.getControl('Branch').value = '~name12/firefox/main'
  >>> user_browser.getControl('Change').click()

Should now be back at the main series page
 
  >>> user_browser.url
  'http://launchpad.dev/firefox/trunk'
  >>> print find_portlet(user_browser.contents, 'Revision control details')
  <div class="portlet">
  ...<a href="http://launchpad.dev/~name12/+branch/firefox/main">Mozilla Firefox Mainline</a>...
  </div>
