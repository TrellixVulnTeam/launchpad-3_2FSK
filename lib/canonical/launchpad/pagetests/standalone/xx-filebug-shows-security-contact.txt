When there is no security contact, the distribution or product
maintainer is notified of the security bug. This ensures that somebody
other than the reporter gets notified.

    >>> user_browser.open("http://launchpad.dev/products/firefox/+filebug")
    >>> print user_browser.contents
    <!DOCTYPE...
    ...The maintainer of...Mozilla Firefox...Sample Person...will be notified...
    ...

    >>> user_browser.open("http://launchpad.dev/distros/ubuntu/+filebug")

    >>> print user_browser.contents
    <!DOCTYPE...
    ...The maintainer of...Ubuntu...Ubuntu Team...will be notified...
    ...

    >>> user_browser.open(
    ...     "http://launchpad.dev/distros/ubuntu/+source/mozilla-firefox/"
    ...     "+filebug")

    >>> print user_browser.contents
    <!DOCTYPE...
    ...The maintainer of...Ubuntu...Ubuntu Team...will be notified...

The bug filing form shows the security contact, when there is
one. Let's set a security contact on each of Mozilla Firefox and Ubuntu
to demonstrate.

    >>> from zope.component import getUtility
    >>> from canonical.database.sqlbase import flush_database_updates
    >>> from canonical.launchpad.interfaces import (
    ...     IProductSet, IDistributionSet, IPersonSet)

(Login, to access various utilities, and set attributes.)

    >>> from canonical.launchpad.ftests import login, logout

    >>> login("foo.bar@canonical.com")

    >>> ubuntu = getUtility(IDistributionSet).getByName("ubuntu")
    >>> mozilla_firefox = getUtility(IProductSet).getByName("firefox")

    >>> ubuntu.security_contact = getUtility(IPersonSet).getByName("no-priv")
    >>> mozilla_firefox.security_contact = getUtility(IPersonSet).getByName(
    ...     "name16")
    >>> flush_database_updates()
    >>> logout()

    >>> user_browser.open("http://launchpad.dev/products/firefox/+filebug")

    >>> print user_browser.contents
    <!DOCTYPE...
    The security contact for...Mozilla Firefox...Foo Bar...will be notified...

    >>> user_browser.open("http://launchpad.dev/distros/ubuntu/+filebug")

    >>> print user_browser.contents
    <!DOCTYPE...
    The security contact for...Ubuntu...No Privileges Person...will be notified...

    >>> user_browser.open(
    ...     "http://launchpad.dev/distros/ubuntu/+source/mozilla-firefox/"
    ...     "+filebug")

    >>> print user_browser.contents
    <!DOCTYPE...
    The security contact for...Ubuntu...No Privileges Person...will be notified...
